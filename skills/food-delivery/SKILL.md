# 🍜 外卖助手技能

**智能外卖推荐 + 快速下单链接生成**

---

## 📋 功能概述

**定位**：不是直接下单（平台不开放API），而是**智能推荐 + 辅助决策 + 快速链接**

### 核心能力

1. **🍴 餐厅推荐** - 基于位置/口味/预算/时间
2. **🔗 快速链接** - 生成美团/饿了么搜索链接
3. **🎲 决策助手** - "吃什么"随机推荐
4. **👥 多人协调** - 群聊点餐统计
5. **📝 订单记录** - 历史订单 + 偏好分析
6. **⏰ 饭点提醒** - 定时提醒吃饭

## 🛠️ 技术实现

### 方案选择

**❌ 不可行方案**：
- 爬虫抓取美团/饿了么（违反ToS，不稳定）
- 官方API（不开放个人用户接口）

**✅ 可行方案**：
1. **智能推荐** - 基于规则 + 用户偏好
2. **链接生成** - 生成平台搜索URL
3. **本地记录** - 存储订单历史
4. **群聊协调** - 统计多人需求

## 🎨 推荐算法

### 数据源
1. **用户偏好** (`user-preferences.json`)
   - 口味偏好：川菜/粤菜/湘菜/西餐...
   - 预算范围：20-50元
   - 忌口：香菜/辣/海鲜...
   - 历史最爱：Top 10 餐厅

2. **餐厅数据库** (`chains.md`)
   - 连锁餐厅：老乡鸡、真功夫、肯德基...
   - 人均价格、距离、评分
   - 推荐菜品

3. **时间因素**
   - 早餐（7-9点）：包子/豆浆/面条
   - 午餐（11-13点）：盖浇饭/快餐
   - 晚餐（17-19点）：正餐/聚餐
   - 夜宵（21-23点）：烧烤/小龙虾

### 推荐流程

```
用户输入 → 解析意图 → 
  ↓
读取用户偏好 → 筛选候选餐厅 →
  ↓
考虑时间/预算/距离 → 排序 →
  ↓
生成推荐 + 链接
```

## ⚠️ 限制说明

1. **无法直接下单** - 需要用户点击链接手动下单
2. **无实时价格** - 价格仅供参考，以平台为准
3. **无店铺数据** - 无法获取实时店铺信息
4. **位置依赖** - 推荐基于用户设置的位置
