```text
$ARGUMENTS
```

在继续之前，你必须先考虑用户输入（如果不为空）。

## 流程（process workflow)

执行 `/ease.specify $ARGUMENTS`

## step2

执行 `/ease.clarify `

## step3

执行 `/ease.plan "参考设计模式文档: ../references/design-patterns.md" `

## step4

执行 `/ease.tasks `

## step5

执行 `/ease.implement `
