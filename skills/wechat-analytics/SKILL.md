# 微信公众号数据分析

## 快速开始

### 获取文章阅读数据

```bash
python scripts/wechat_analytics.py article --date 2026-02-16
```

### 获取用户数据

```bash
python scripts/wechat_analytics.py user --days 7
```

### 生成选题建议

```bash
python scripts/wechat_analytics.py suggest
```

## 主要功能

### 1. 文章数据分析
- 每日阅读数据
- 阅读来源分布
- 分享转发数据

### 2. 用户数据分析
- 用户增长趋势
- 累计用户数据
- 用户属性分布

### 3. 选题优化
- 基于历史数据推荐热门主题
- 识别用户兴趣偏好
- 生成选题建议报告

## 数据用途

1. **了解用户** - 知道用户喜欢什么内容
2. **优化选题** - 根据阅读量调整方向
3. **改进标题** - 分析哪些标题更吸引人
4. **发布时间** - 找出最佳发布时间

## 注意事项

- 数据统计有 1 天延迟
- 部分数据需要认证账号才能获取
- 数据仅供参考，内容质量是关键
