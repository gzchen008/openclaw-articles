OpenClaw v2026.3.23 更新：浏览器稳定性大幅提升

发布时间：2026年3月24日

🚀 本次更新亮点

  • 浏览器连接更稳定，macOS Chrome 附加流程优化
  • ClawHub 技能商店登录问题修复
  • 消息工具支持飞书文件/图片附件发送
  • Mistral 模型配置自动修复
  • 代理环境下的 OAuth 登录支持

🔧 浏览器与网络

Chrome 浏览器附加优化
  • 修复 macOS Chrome 附加超时问题：修复了用户配置文件附加时的超时和重复授权问题，现在浏览器附加更加流畅
  • 修复 Linux 无头浏览器二次启动问题：解决了较慢的 Linux 环境下浏览器第二次启动失败的问题

🛒 ClawHub 技能商店

登录与认证修复
  • 修复 macOS 登录状态保持：修复了 macOS 上 ClawHub 凭据读取问题，现在技能浏览和安装会正确使用登录状态
  • 修复 认证降级问题：解决了技能浏览时意外降级到未认证模式导致的 429 错误

💬 消息工具增强

Discord、Slack、飞书
  • 修复 Discord 组件和 Slack 块可选：恢复了 components 和 blocks 的可选性
  • 修复 飞书文件/图片发送：修复了飞书消息工具发送文件和图片附件失败的问题
  • 修复 Telegram 消息优化：新增 asDocument 参数，改进了消息防重复机制
  • 修复 Discord 权限命令：特权命令现在会返回明确的未授权提示

🤖 模型与代理

模型配置优化
  • 修复 Mistral 模型配置：降低了 Mistral 最大 token 默认值，openclaw doctor --fix 可自动修复旧配置
  • 修复 OpenRouter 定价刷新：修复了 openrouter/auto 定价递归刷新问题
  • 修复 代理环境 OAuth：OpenAI Codex 和 MiniMax 的 OAuth 登录现在支持代理环境
  • 修复 web_search 提供者：使用实际配置的搜索提供者而非默认值

⚡ 子代理与技能

运行时改进
  • 修复 子代理超时误报：快速完成的子代理不再被错误报告为超时
  • 修复 技能 API Key 解析：修复了嵌入技能的 SecretRef 解析问题
  • 修复 Anthropic thinking 块顺序：保持了思考块的正确顺序

🔐 安全与稳定性

安全加固
  • 修复 Shell 命令审批：加强了 shell 包装器的安全检查
  • 修复 Canvas 路由鉴权：Canvas 路由现在需要认证
  • 修复 网关锁冲突：修复了 launchd 和 systemd 下的崩溃循环问题
  • 修复 插件配置容错：过期的插件 ID 现在是警告而非致命错误

📦 安装与发布

  • 修复 npm 包完整性：发布的 npm 包现在包含所有必要的插件和 Control UI 资源
  • 修复 Matrix 插件：修复了 Matrix 插件启动时的重复属性错误
  • 修复 内存插件：LanceDB 内存插件现在可以在全局 npm 安装后正常工作

📥 如何更新

推荐更新命令：

npm install -g openclaw@latest

或使用：openclaw update
