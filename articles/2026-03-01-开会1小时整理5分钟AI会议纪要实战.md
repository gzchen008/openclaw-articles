## 📋 前置条件

在开始之前，请确保：

- 已安装 OpenClaw（没有的话看上一篇安装指南）
- 有一个可用的 ASR 服务（语音转文字）
- 有一点耐心（配置大约 15 分钟）

## 🔧 第一步：配置语音转文字

OpenClaw 本身不做语音识别，但可以对接各种 ASR 服务。推荐几个方案：

<p style="margin: 4px 0; padding-left: 20px;">• <strong style="font-weight: 600;">OpenAI Whisper</strong>：免费、效果好，本地运行</p>
<p style="margin: 4px 0; padding-left: 20px;">• <strong style="font-weight: 600;">阿里云 ASR</strong>：中文效果好，有免费额度</p>
<p style="margin: 4px 0; padding-left: 20px;">• <strong style="font-weight: 600;">讯飞听见</strong>：中文最准，需要付费</p>

这里以 Whisper 为例，配置一个简单的转写服务：

<section style="background: #f6f8fa; padding: 12px; border-radius: 6px; margin: 8px 0;">
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">pip install openai-whisper</p>
</section>

然后在 OpenClaw 中添加一个技能来调用它：

<section style="background: #f6f8fa; padding: 12px; border-radius: 6px; margin: 8px 0;">
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;"># skills/transcribe/SKILL.md</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">name: transcribe</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">description: 转写音频文件为文字</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">exec: python scripts/transcribe.py $audio_path</p>
</section>

## 🤖 第二步：创建会议纪要模板

AI 整理会议纪要需要明确的格式要求。在 OpenClaw 的记忆文件中添加模板：

<section style="background: #f6f8fa; padding: 12px; border-radius: 6px; margin: 8px 0;">
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;"># 会议纪要模板</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">## 会议信息</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">- 时间：[自动填充]</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">- 参与人：[从对话中提取]</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">- 主题：[总结]</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">## 关键决策</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">[列出所有确定的决策]</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">## 待办事项</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">- [ ] 任务1 @负责人 截止时间</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">- [ ] 任务2 @负责人 截止时间</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">## 下次会议</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">[时间和议题]</p>
</section>

这样 AI 就知道该按什么结构来整理了。

## ⏰ 第三步：配置自动化流程

最理想的方式是：会议开始时发一条消息，AI 自动开始记录；会议结束时再发一条，AI 自动生成纪要。

用 OpenClaw 的 Cron 功能可以实现：

<section style="background: #f6f8fa; padding: 12px; border-radius: 6px; margin: 8px 0;">
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;"># 每周五下午 3 点提醒生成周会纪要</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">cron:</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">  - name: weekly-meeting-summary</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">    schedule: "0 15 * * 5"</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">    task: "读取本周会议记录，生成周会纪要"</p>
</section>

也可以在对话中直接说：

<p style="margin: 4px 0; padding-left: 20px;">• 「帮我整理今天的会议内容」</p>
<p style="margin: 4px 0; padding-left: 20px;">• 「生成周会纪要」</p>
<p style="margin: 4px 0; padding-left: 20px;">• 「提取刚才说的待办事项」</p>

## 💡 实战技巧

### 1. 会前准备

开会前，把议程发给 OpenClaw：

<section style="background: #f6f8fa; padding: 12px; border-radius: 6px; margin: 8px 0;">
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">今天的周会议程：</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">1. 上周数据回顾</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">2. 项目进度同步</p>
  <p style="margin: 0; font-family: monospace; font-size: 14px; color: #24292e;">3. 本周重点讨论</p>
</section>

AI 会根据议程来组织纪要结构。

### 2. 会中实时辅助

如果会议在 Discord 或 Telegram 进行，OpenClaw 可以实时参与：

<p style="margin: 4px 0; padding-left: 20px;">• 自动记录关键发言</p>
<p style="margin: 4px 0; padding-left: 20px;">• 提醒跑题（「当前讨论偏离议程第 2 项」）</p>
<p style="margin: 4px 0; padding-left: 20px;">• 追踪时间（「讨论还有 5 分钟」）</p>

### 3. 会后自动分发

纪要生成后，可以自动发送到指定渠道：

<p style="margin: 4px 0; padding-left: 20px;">• 发到团队微信群</p>
<p style="margin: 4px 0; padding-left: 20px;">• 发到 Slack 频道</p>
<p style="margin: 4px 0; padding-left: 20px;">• 发到 Notion 页面</p>
<p style="margin: 4px 0; padding-left: 20px;">• 发邮件给参会人</p>

## ⚠️ 注意事项

<p style="margin: 4px 0; padding-left: 20px;">• <strong style="font-weight: 600;">隐私保护</strong>：敏感会议建议关闭自动转写，手动整理</p>
