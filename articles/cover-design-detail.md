# 【封面图】制作详情 - 直接可用

## 📐 规格
- **尺寸**：900 x 383 像素（公众号封面标准）
- **风格**：科技感 + 金融专业感
- **主色调**：深蓝 + 金色

---

## 🎨 Canva 制作步骤（5分钟搞定）

### Step 1: 创建画布
1. 打开 canva.com
2. 点击「创建设计」→「自定义大小」
3. 输入：宽 900px，高 383px
4. 点击「创建新设计」

### Step 2: 设置背景
1. 点击左侧「元素」→「形状」→ 选择矩形
2. 拉伸填满整个画布
3. 点击颜色 → 选择「渐变色」
4. 设置渐变：
   - 左/上：#0f172a（深蓝黑）
   - 右/下：#1e3a5f（深蓝）
   - 角度：135度

### Step 3: 添加主标题（直接复制）

**点击「文字」→「添加标题」→ 输入：**

```
我用AI选股
```

**设置：**
- 字体：思源黑体 Bold（或 阿里巴巴普惠体 Bold）
- 字号：60px
- 颜色：#ffffff（白色）
- 位置：左侧，垂直居中偏上

---

**再添加一个标题：**

```
3个月跑赢大盘20%
```

**设置：**
- 字体：同上
- 字号：48px
- 颜色：#f5a623（金色）
- 位置：主标题下方，左侧对齐

---

**再添加副标题：**

```
附完整系统搭建教程
```

**设置：**
- 字体：思源黑体 Regular
- 字号：24px
- 颜色：#94a3b8（浅灰）
- 位置：金色标题下方

### Step 4: 添加装饰元素

**右侧添加AI机器人图标：**
1. 点击「元素」→ 搜索 "robot"
2. 选择：白色/线条风格的机器人或AI大脑图标
3. 调整大小：约占画面右侧 1/3
4. 颜色：#ffffff（白色）或 #f5a623（金色）

**添加K线背景（半透明）：**
1. 点击「元素」→ 搜索 "stock chart" 或 "trading"
2. 选择：上升趋势的折线图或K线图
3. 调整：透明度 20-30%
4. 位置：铺满背景，在文字后面
5. 颜色：#3b82f6（蓝色）

**添加收益率标签：**
1. 点击「元素」→「形状」→ 圆角矩形
2. 设置颜色：#f5a623（金色）
3. 添加文字：「+20%」
   - 字号：36px
   - 颜色：#0f172a（深蓝）
   - 加粗
4. 位置：右上角

### Step 5: 添加向上箭头
1. 点击「元素」→ 搜索 "arrow up"
2. 选择：粗线条向上箭头
3. 颜色：#10b981（绿色）
4. 位置：+20% 标签旁边
5. 大小：适中，醒目但不突兀
