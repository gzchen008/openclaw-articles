OpenClaw v2026.3.22 重大更新：ClawHub 插件市场、Chrome MCP、安全加固

OpenClaw 刚刚发布 v2026.3.22 版本！这是继 v2026.3.13 之后的重大更新，包含 ClawHub 插件市场、Chrome MCP 集成、安全加固等重磅功能，以及 200+ 项改进和修复。

🚀 核心新功能

1. ClawHub 插件市场

• 原生支持：openclaw skills search|install|update

• 插件安装：openclaw plugins install clawhub:<package>

• 自动更新：支持版本追踪和自动更新

• 市场发现：更容易找到和安装社区插件

🔗 文档：ClawHub

2. Chrome MCP 集成

• 移除旧版：删除 Chrome extension relay 路径

• 现有会话：支持 existing-session 和 user 配置

• 跨浏览器：支持 Brave、Edge 等 Chromium 浏览器

• 用户数据目录：可配置 browser.profiles.<name>.userDataDir

3. 多模型支持增强

• OpenAI：默认模型升级到 gpt-5.4

• MiniMax：新增 M2.7 和 M2.7-highspeed 模型

• xAI：同步 Grok 模型目录和定价

• Anthropic Vertex：支持 Claude via Google Vertex AI

• Chutes：新增 Chutes 提供商插件

4. Web 工具集成

• Exa：新增 Exa 搜索插件

• Tavily：新增 tavily_search 和 tavily_extract 工具

• Firecrawl：新增 Firecrawl 搜索和抓取插件

🛡️ 安全加固

⚠️ 重要安全修复

• Windows 媒体安全：阻止远程 file:// URL 和 UNC 网络路径

• 网络发现：修复 Bonjour/DNS-SD 服务端点验证

• 设备配对：绑定 iOS setup codes 到正确的节点配置

• Nostr 安全：强制入站 DM 策略，添加预加密速率限制

• Exec 审批：转义 Hangul filler 代码点，防止视觉欺骗

• Safe bins：从默认 allowlist 移除 jq

📱 渠道更新

Telegram

• 自定义 Bot API：支持自定义 Bot API 端点

• Topic 自动命名：首次消息时自动重命名 DM forum topics

• 主题编辑：新增 topic-edit 操作

• 静默错误回复：可选的静默错误消息发送

飞书

• 交互卡片：新增结构化审批和快速操作启动器

• ACP 绑定：支持当前会话 ACP 和子代理会话绑定

• 流式推理：支持推理 token 流式渲染

• 身份感知：卡片头部和页脚显示身份信息

Matrix

• 新插件：基于官方 matrix-js-sdk 的新 Matrix 插件

• Bot 交互：支持 allowBots 房间策略

• 私有网络：支持私有/内部 homeservers

⚡ 性能优化

• 启动速度：Gateway 启动不再重新编译 TypeScript，WhatsApp 冷启动回到秒级

• CLI 延迟加载：减少内存占用和启动延迟

• 模型预热：启动时预热主模型，避免首条消息失败

• 会话缓存：自动清理过期会话缓存条目

🔧 其他改进

• Agent 超时：默认超时从 600s 提升到 48h

• 插件市场：支持 Claude marketplace 注册表

• Sandbox：新增 SSH sandbox 后端和 OpenShell 后端

• Android：新增 Call Log 和 SMS 搜索权限支持

• Control UI：新增展开到 Canvas 按钮，统一主题边框半径

📦 Breaking Changes（重要）

⚠️ 如果你正在使用以下功能，升级前请注意：

• Chrome extension relay：已移除，改用 existing-session 或 user

• Plugin SDK：openclaw/extension-api 已移除，改用 openclaw/plugin-sdk/*

• Config：CLAWDBOT_* 和 MOLTBOT_* 环境变量已移除，改用 OPENCLAW_*

• State：~/.moltbot 状态目录已移除，改用 ~/.openclaw

• Matrix：旧版 Matrix 插件已替换，需要迁移配置

🚀 如何升级

升级命令：

openclaw update

或者从 GitHub main 分支安装：

openclaw update --tag main
