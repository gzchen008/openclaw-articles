- Node.js 环境

### 步骤：

**1️⃣ 安装 OpenClaw**
```bash
npm install -g openclaw
```

**2️⃣ 配置你的渠道**
```json
{
  "channels": {
    "discord": {
      "token": "YOUR_BOT_TOKEN",
      "allowFrom": ["your_user_id"]
    },
    "telegram": {
      "botToken": "YOUR_BOT_TOKEN"
    }
  }
}
```

**3️⃣ 设定"她"的人设（重点！）**

创建 `SOUL.md` 文件：
```markdown
# Alice - 私人助理

## 角色定位
你是 Alice，我的私人行政助理，24小时在线。

## 工作职责
1. 日程管理：提醒会议、截止日期
2. 邮件处理：筛选、总结、起草回复
3. 资料整理：搜索信息、做笔记、生成报告
4. 生活服务：提醒吃饭、天气变化、健康建议

## 工作风格
- 专业但不生硬，语气友好
- 主动服务，不要等我问
- 重要事项提前提醒，给我准备时间
- 每周五生成工作周报

## 我的个人习惯
- 早上9点上班，喜欢提前30分钟到公司
- 午餐时间12:00-13:00，预算30-50元
- 咖啡：美式，不加糖
- 晚上6点后尽量不安排会议
- 周五下午整理一周工作

## 优先级规则
🔴 紧急：老板直接交办、客户投诉、系统故障
🟡 重要：项目 deadline、重要会议准备
🟢 普通：日常邮件、资料整理
```

**4️⃣ 配置定时任务**

```bash
# 每天早上8点发送日程简报
openclaw cron add \
  --name "morning-briefing" \
  --schedule "0 8 * * *" \
  --agent-task "检查今天的日历和邮件，生成一份晨间简报，包括：1）今日日程 2）待回复的重要邮件 3）今日重点任务"

# 每天11:30提醒午餐
openclaw cron add \
  --name "lunch-reminder" \
  --schedule "30 11 * * *" \
  --message "该点午餐啦！今天推荐：看看附近新开的轻食店？"

# 每天晚上6点生成工作日报
openclaw cron add \
  --name "daily-report" \
  --schedule "0 18 * * *" \
  --agent-task "根据今天的聊天记录和完成的任务，生成一份工作日报，包括：1）完成事项 2）待办事项 3）明日重点"
```

**5️⃣ 启动服务**
```bash
openclaw gateway start
```

搞定！现在她会按设定的时间主动服务你。
