程序员必备：Slack 里的 AI 编程助手

你有没有想过，Slack 频道里有个 24 小时在线的 AI 编程助手，能帮你写代码、查 Bug、解释技术概念？

今天教你用 OpenClaw，把 AI 助手装进 Slack。

━ 🎯 最终效果 ━

配置完成后，你的 Slack AI 助手可以：

✅ 自动回答技术问题

✅ 帮你写代码片段

✅ 解释报错信息

✅ 做代码审查

✅ 发送定时提醒

━ 📝 准备工作 ━

1. 创建 Slack App

1️⃣ 访问 Slack API

2️⃣ 点击 Create New App

3️⃣ 选择 From scratch

4️⃣ 填写 App 名称和工作区

2. 配置权限

在 OAuth & Permissions 页面添加以下权限：

Bot Token Scopes：

• app_mentions:read - 读取 @提及

• chat:write - 发送消息

• channels:history - 读取频道历史

• groups:history - 读取私聊历史

• im:history - 读取私信历史

3. 安装到工作区

1️⃣ 点击 Install to Workspace

2️⃣ 授权权限

3️⃣ 复制 Bot User OAuth Token（以 xoxb- 开头）

4. 获取 Signing Secret

在 Basic Information 页面找到 Signing Secret，复制备用。

━ ⚙️ 配置 OpenClaw ━

1. 添加 Slack 渠道

编辑 OpenClaw 配置文件：

{

"channels": {

"slack": {

"enabled": true,

"botToken": "xoxb-YOUR_BOT_TOKEN_HERE",

"signingSecret": "YOUR_SIGNING_SECRET_HERE"

}

}

}

2. 启用 Event Subscriptions

在 Slack App 设置中：

1️⃣ 进入 Event Subscriptions

2️⃣ 开启 Enable Events

3️⃣ 填写 Request URL：https://your-gateway-url/slack/events

4️⃣ 订阅事件：app_mention、message.channels

3. 重启 Gateway

openclaw gateway restart

4. 测试连接

在 Slack 频道里 @你的 Bot，看是否收到回复。

━ 🤖 进阶功能 ━

在频道里发代码，让 AI 帮你 Review：

@Bot 请帮我审查这段代码：

```python

def process_data(data):

return data.strip().lower()

```

定时提醒

用 Cron 发送每日站会提醒：

{

"name": "daily-standup",

"schedule": "0 10 * * 1-5",

"action": "message",

"channel": "slack",

"target": "#general",

"message": "🔔 站会时间到！请分享：昨天做了什么、今天计划做什么、有什么阻碍。"

}

━ ⚠️ 常见问题 ━

Q: Bot 不回复 @提及？

检查以下几点：

1. Bot Token 和 Signing Secret 是否正确

2. Event Subscriptions 是否启用

3. Request URL 是否可访问

4. Bot 是否在频道里（需要邀请加入）

Q: 如何让 Bot 响应所有消息（不只是 @提及）？

1. 在 Event Subscriptions 添加 message.channels 事件

2. 在 OAuth & Permissions 添加 channels:history 权限

3. 在频道设置中允许 Bot 读取消息

Q: 如何限制 Bot 只在特定频道工作？

在 OpenClaw 配置中添加白名单：

{

"channels": {

"slack": {

"enabled": true,

"allowedChannels": ["C12345678", "C87654321"]

}

}

}

Q: 如何获取频道 ID？

1. 右键点击频道名 → Copy Link

2. 链接最后一段就是频道 ID（如 C12345678）
