# OpenClaw v2026.3.11 发布：重要安全更新 + Ollama 一键配置

> 发布时间：2026年3月12日

OpenClaw 最新版本 **v2026.3.11** 正式发布！本次更新包含一个重要的**安全修复**，以及多项实用新功能。

---

## ⚠️ 重要安全更新

### WebSocket 跨站劫持漏洞修复

本次更新修复了一个跨站 WebSocket 劫持漏洞（CVE: GHSA-5wcw-8jjv-m286）。

**影响范围**：使用 `trusted-proxy` 模式的 Gateway 部署

**风险**：未受信任的来源可能获得 `operator.admin` 访问权限

**建议**：**请尽快升级到此版本！**

---

## ✨ 新功能亮点

### 1. Ollama 一键配置向导

现在可以直接在 OpenClaw 中完成 Ollama 设置：
- 支持 **Local** 和 **Cloud + Local** 两种模式
- 浏览器端云账号登录
- 智能模型推荐
- 云端模型自动跳过本地拉取

```bash
openclaw onboarding
```

### 2. OpenCode Go 提供商

新增 OpenCode Go 原生支持：
- 统一的 Zen 和 Go 设置体验
- 一个 API Key 管理两个提供商
- 保留运行时分离

### 3. iOS Home Canvas 全新改版

- 新增欢迎界面，实时显示 Agent 状态
- 连接/重连/前台返回时自动刷新
- 浮动控制改为底部工具栏
- 适配小屏手机

### 4. macOS Chat UI 模型选择器

- 可视化模型选择界面
- 思考级别选择持久化
- 提供商感知的会话模型同步

### 5. Hunter Alpha & Healer Alpha 模型

临时新增两个免费 stealth 模型（OpenRouter 用户可用）：
- `hunter-alpha` - 高性能推理
- `healer-alpha` - 均衡型

⚠️ 预计可用期：约一周

### 6. 多模态记忆索引

支持图片和音频的语义搜索：
- 使用 Gemini `gemini-embedding-2-preview`
- 可配置输出维度
- 自动重新索引

## 🐛 重要修复

### 模型兼容性

- **GLM-5 / DeepSeek**：过滤内部控制标记，不再显示 `<｜...｜>` 乱码
- **Kimi Coding**：恢复原生工具调用格式
- **Azure OpenAI**：修复多轮任务 400 错误
- **Venice / Poe**：识别余额不足错误，触发模型回退

### 消息平台

- **Telegram**：长消息 HTML 分块发送
- **Discord**：修复消息分块问题
- **飞书**：本地图片自动上传修复

### 网关稳定性

- iOS 前台恢复立即重连
- macOS LaunchAgent 重启修复
- 配置验证错误提示优化
