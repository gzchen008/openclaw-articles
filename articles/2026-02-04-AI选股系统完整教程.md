### 2️⃣ 因子计算：量化分析的核心

> **【配图建议7】**  
> 制作一张「5大因子雷达图」：
> - 价值因子（蓝色）
> - 成长因子（绿色）
> - 质量因子（紫色）
> - 动量因子（橙色）
> - 情绪因子（红色）
> 每个维度标注典型指标，如PE、ROE、RSI等

**什么是因子？**  
简单说，就是选股的「标准」。比如：PE低、成长快、资金流入……

**我的5大因子体系：**

```python
# 因子计算示例
factors = {
    # 价值因子：便宜才是硬道理
    'PE': 市盈率,
    'PB': 市净率,
    '股息率': 年度分红/股价,
    
    # 成长因子：买未来，不是买过去
    '营收增长率': (今年营收-去年营收)/去年营收,
    '净利润增长率': 同上,
    
    # 质量因子：好公司才能长期持有
    'ROE': 净资产收益率,
    '毛利率': (收入-成本)/收入,
    
    # 动量因子：趋势是你的朋友
    '20日涨幅': 近20天涨跌幅,
    'RSI': 相对强弱指标,
    
    # 情绪因子：别人恐慌我贪婪
    '资金流入': 主力净流入额,
    '舆情得分': AI分析新闻情绪
}
```

**💡 关键**：不要追求完美的因子，先跑起来再优化。

> **【配图建议8】**

**财报解读自动化：**

```python
from openai import OpenAI

client = OpenAI()

def analyze_report(stock_code):
    # 获取财报数据
    report = get_financial_report(stock_code)
    
    prompt = f"""
    你是一位资深证券分析师，请分析以下财报：
    
    公司：{report['name']}
    营收：{report['revenue']}亿元（同比{report['revenue_growth']}%）
    净利润：{report['profit']}亿元（同比{report['profit_growth']}%）
    ROE：{report['roe']}%
    
    请输出：
    1. 核心亮点（如果有）
    2. 主要风险（如果有）
    3. 投资建议（买入/观望/回避）
    """
    
    response = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": prompt}]
    )
    
    return response.choices[0].message.content
```

**成本**：每次分析约 $0.01（人民币7分钱），分析100只股票成本不到1块钱。

### 4️⃣ 策略选股：从5000只中挑出10只

> **【配图建议9】**  
> 流程图展示选股过程：
> 5000只股票 → 过滤（市值、PE等）→ 计算因子得分 → 加权排序 → TOP10
> 用漏斗形状表示筛选过程，每个阶段显示剩余股票数量

**多因子打分模型：**

```python
def select_stocks():
    # 1. 获取全市场股票
    all_stocks = get_all_stocks()
    
    # 2. 计算所有因子
    for stock in all_stocks:
        stock['value_score'] = calculate_value_score(stock)
        stock['growth_score'] = calculate_growth_score(stock)
        stock['quality_score'] = calculate_quality_score(stock)
        stock['momentum_score'] = calculate_momentum_score(stock)
    
    # 3. 综合打分（加权平均）
    for stock in all_stocks:
        stock['total_score'] = (
            stock['value_score'] * 0.3 +
            stock['growth_score'] * 0.3 +
            stock['quality_score'] * 0.2 +
            stock['momentum_score'] * 0.2
        )
    
    # 4. 排序，取前10名
    top_stocks = sorted(all_stocks, 
                       key=lambda x: x['total_score'], 
                       reverse=True)[:10]
    
    return top_stocks
```

**我的选股标准：**
- 市值 > 50亿（避免流动性风险）
- PE < 50（避免过高估值）
- 近3年ROE > 10%（盈利能力）
- 近20日资金净流入（有主力关注）

```python
# 1. 仓位管理
def position_sizing(signal_strength):
    """根据信号强度决定仓位"""
    if signal_strength > 90:
        return 0.15  # 15%仓位
    elif signal_strength > 80:
        return 0.10  # 10%仓位
    else:
        return 0.05  # 5%仓位

# 2. 止损机制
def check_stop_loss(stock):
    """检查是否触发止损"""
    current_price = get_price(stock['code'])
    buy_price = stock['buy_price']
    
    # 固定止损：-8%
    if (current_price - buy_price) / buy_price < -0.08:
        return True, "触发固定止损(-8%)"
    
    # 移动止损：跌破20日均线
    ma20 = calculate_ma(stock['code'], 20)
    if current_price < ma20:
        return True, "跌破20日均线"
    
    return False, ""

# 3. 止盈机制
def check_take_profit(stock):
    """检查是否触发止盈"""
    current_price = get_price(stock['code'])
    buy_price = stock['buy_price']
    
    profit_pct = (current_price - buy_price) / buy_price
    
    # 目标止盈：+20%
    if profit_pct > 0.20:
        return True, "达到目标收益(+20%)"
    
    return False, ""
```

**💡 核心原则**：任何一只股票，最多亏8%；单只股票仓位不超过15%。

## ⚠️ 避坑指南（血的教训）

> **【配图建议13】**  > 四格漫画风格，每个格子一个「坑」：
> 1. 过拟合：回测完美曲线 vs 实盘暴跌曲线
> 2. 交易成本：频繁交易被手续费吃掉
> 3. 黑天鹅：突发新闻（疫情图标）导致系统失效
> 4. 实盘测试：新策略直接实盘亏钱
> 底部大标题：「这些坑我都踩过，别学我」

### 坑1：过度优化（过拟合）
**问题**：回测时表现极好，实盘就亏。
**解决**：
- 用最近3年数据回测，不要只用牛市数据
- 留出一部分数据做「样本外测试」
- 参数不要调得太精细，留有余地

### 坑2：忽视交易成本
**问题**：频繁交易，手续费吃掉利润。
**解决**：
- 每次交易至少持有5天以上
- 计算收益时扣除印花税（卖出0.1%）+ 佣金

### 坑3：全部自动化，完全不看
**问题**：遇到黑天鹅事件（如疫情），系统无法识别。
**解决**：
- 每天花5分钟看下持仓新闻
- 重大事件（如业绩预告）手动确认

### 坑4：用实盘测试新策略
**问题**：新策略未经验证，直接亏钱。
**解决**：
- 先用模拟盘跑3个月
- 验证有效后再上实盘
- 实盘初期只用小资金（10%仓位）
