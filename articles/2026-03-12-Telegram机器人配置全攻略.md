## 第一步：创建 Telegram 机器人

### 1.1 找到 BotFather

在 Telegram 搜索 `@BotFather`，这是官方的机器人管理工具。

### 1.2 创建新机器人

发送 `/newbot`，然后按提示操作：

1. 输入机器人名称（比如：MyAIAssistant）
2. 输入机器人用户名（必须以 `bot` 结尾，比如：my_ai_assistant_bot）

BotFather 会返回一个 **Token**，格式类似：

```
123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

**⚠️ 重要**：这个 Token 要保密！泄露了别人就能控制你的机器人。

---

## 第二步：获取你的 Chat ID

机器人需要知道"你是谁"才能给你发消息。

### 2.1 私聊方式

1. 在 Telegram 搜索 `@userinfobot`
2. 发送任意消息
3. 它会返回你的 Chat ID（一串数字）

### 2.2 群聊方式

1. 把 `@userinfobot` 拉进群
2. 在群里发消息
3. 它会返回群的 Chat ID（负数）

---

## 第三步：配置 OpenClaw

### 3.1 编辑配置文件

打开 OpenClaw 配置文件 `~/.openclaw/openclaw.json`，添加 Telegram 配置：

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "YOUR_BOT_TOKEN_HERE",
      "allowedChats": [123456789, -1001234567890]
    }
  }
}
```

**参数说明**：
- `botToken`：刚才从 BotFather 获取的 Token
- `allowedChats`：允许机器人响应的 Chat ID 列表（私聊和群聊都可以）

### 3.2 重启 OpenClaw

```bash
openclaw gateway restart
```

## 第五步：进阶配置

### 5.1 设置触发关键词

你可以配置机器人只在特定关键词出现时响应：

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "YOUR_BOT_TOKEN_HERE",
      "allowedChats": [123456789],
      "triggers": {
        "prefix": ["/ai", "@my_bot"],
        "keywords": ["帮忙", "问题"]
      }
    }
  }
}
```

这样设置后：
- 发送 `/ai 今天天气怎么样？` → AI 回复
- 发送 `@my_bot 帮我写个代码` → AI 回复
- 发送 `帮忙看看这个` → AI 回复
- 发送普通消息 → AI 不响应（避免刷屏）

### 5.2 配置群组权限

控制机器人在不同群的行为：

```json
{
  "groups": {
    "-1001234567890": {
      "name": "工作群",
      "mode": "mention",
      "adminOnly": false
    },
    "-1009876543210": {
      "name": "客服群",
      "mode": "all",
      "adminOnly": true
    }
  }
}
```

**mode 选项**：
- `all`：响应所有消息
- `mention`：只响应 @机器人 的消息
- `admin`：只响应管理员的消息

## ⚠️ 常见问题

### Q: 机器人不回复消息？

检查这几项：
1. Bot Token 是否正确
2. Chat ID 是否在 `allowedChats` 列表里
3. 是否先发送了 `/start` 命令
4. OpenClaw 是否正常运行（`openclaw status`）

### Q: 群里消息太多，机器人刷屏？

设置 `mode: "mention"`，只有 @机器人 时才回复。

### Q: 如何让机器人只回复管理员？

设置 `adminOnly: true`，并配置管理员 ID 列表。

### Q: Token 泄露了怎么办？

找 @BotFather 发送 `/revoke`，重新生成 Token。
