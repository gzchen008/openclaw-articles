## 🎯 最终效果

配置完成后，你的 Telegram 机器人可以：

- 自动回复用户消息（客服场景）
- 在群里自动应答（社区管理）
- 定时推送消息（提醒、日报）
- 执行自定义命令（查天气、搜资料）

## 📋 前置条件

开始之前，你需要准备：

1. 一个 Telegram 账号
2. 已安装 OpenClaw（如果还没安装，看这篇[安装指南](#)）
3. 基本的命令行操作能力

## 🔧 第一步：创建 Telegram Bot

### 1. 找到 BotFather

在 Telegram 搜索 `@BotFather`，这是官方的机器人管理工具。

### 2. 创建新机器人

发送命令：

```
/newbot
```

然后按提示操作：

1. 输入机器人名称（比如：`我的AI助手`）
2. 输入机器人用户名（必须以 `bot` 结尾，比如：`my_ai_helper_bot`）

创建成功后，BotFather 会给你一个 **Token**，长得这样：

```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**⚠️ 重要：这个 Token 要保密，相当于机器人的密码！**

## ⚙️ 第二步：配置 OpenClaw

### 1. 打开配置文件

找到 OpenClaw 的配置文件 `~/.openclaw/openclaw.json`，添加 Telegram 配置：

```json
{
  "telegram": {
    "enabled": true,
    "token": "你的机器人Token",
    "allowedUsers": ["你的用户ID"],
    "allowedGroups": ["群组ID"]
  }
}
```

### 2. 获取你的用户 ID

在 Telegram 搜索 `@userinfobot`，发送任意消息，它会返回你的用户 ID（是一串数字）。

### 3. 获取群组 ID（可选）

把机器人加入群组后，在群里发送消息，OpenClaw 的日志会显示群组 ID。

## 🚀 第三步：启动机器人

### 1. 重启 OpenClaw

```bash
openclaw restart
```

### 2. 测试机器人

在 Telegram 给你的机器人发送消息：

```
你好
```

如果一切正常，OpenClaw 会自动回复！

## 💡 第四步：配置群组权限（可选）

如果你想让机器人在群里工作，需要配置群组权限：

### 1. 把机器人加入群组

在群组设置里，点击「添加成员」，搜索你的机器人用户名。

### 2. 设置管理员权限（推荐）

给机器人设置管理员权限，这样它可以：

- 删除违规消息
- 管理群成员
- 置顶重要通知

### 3. 配置群组白名单

在 `openclaw.json` 里添加群组 ID：

```json
{
  "telegram": {
    "allowedGroups": ["-1001234567890"]
  }
}
```

## 🎨 进阶：自定义机器行为

### 设置欢迎消息

当新成员加入群组时，自动发送欢迎消息：

```json
{
  "telegram": {
    "welcomeMessage": "欢迎加入！我是 AI 助手，有问题随时问我 👋"

```

### 设置自动回复

针对常见问题，设置关键词自动回复：

```json
{
  "telegram": {
    "autoReplies": {
      "价格": "我们的产品价格请查看官网：xxx.com",
      "客服": "客服工作时间：9:00-18:00"
    }
  }
}
```

## ❓ 常见问题

### Q: 机器人不回复消息？

检查以下几点：

1. Token 是否正确
2. 用户 ID 是否在白名单里
3. OpenClaw 是否正常运行（`openclaw status`）
4. 查看日志是否有错误（`openclaw logs`）

### Q: 机器人在群里不工作？

确认：

1. 机器人已加入群组
2. 群组 ID 在 `allowedGroups` 里
3. 机器人有发言权限

### Q: 如何让机器人只回复特定消息？

使用关键词过滤：

```json
{
  "telegram": {
    "replyPrefix": "/ai",
    "replyOnlyWhenMentioned": true
  }
}
```

这样机器人只在被 @ 或收到 `/ai` 命令时才回复。
