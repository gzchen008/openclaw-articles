⚙️ 如何配置？

Step 1：设置笔记存储位置

在 OpenClaw 配置中指定：

{

"knowledge": {

"storage": "/path/to/obsidian/vault",

"auto_tag": true,

"daily_review": "09:00"

}

}

Step 2：配置定时任务

设置每日回顾：

openclaw cron add \

--name "daily-review" \

--schedule "0 9 * * *" \

--action "push_daily_notes"

Step 3：连接微信/Discord

在微信或 Discord 里直接发给 OpenClaw，它会自动处理。
