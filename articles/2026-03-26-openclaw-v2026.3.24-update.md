# OpenClaw 2026.3.24 重磅更新：Teams 官方 SDK、OpenAI 兼容性增强、一键安装依赖

> 从 2026.3.13 升级到 2026.3.24，这个版本带来了多项重大改进！

---

## 🎯 核心亮点

### 1️⃣ OpenAI 兼容性大幅增强

Gateway 现在支持更多 OpenAI API 端点：

- **新增 `/v1/models` 端点** - 查询可用模型列表
- **新增 `/v1/embeddings` 端点** - 支持向量嵌入，RAG 应用友好
- **模型覆盖转发** - `/v1/chat/completions` 和 `/v1/responses` 支持显式模型覆盖

这意味着你的 RAG 应用、第三方客户端可以更无缝地对接 OpenClaw Gateway！

### 2️⃣ Microsoft Teams 全面升级

迁移到 **官方 Teams SDK**，带来 AI Agent 最佳实践：

- ✨ 流式 1:1 回复 - 打字机效果，体验更流畅
- 🎯 欢迎卡片 + 提示词快捷入口
- 💬 消息编辑和删除支持
- 🤖 原生 AI 标签显示
- 📊 反馈/反思机制

### 3️⃣ Skills 一键安装依赖

Control UI 和 CLI 现在可以**自动检测并安装技能依赖**：

```
openclaw skills info <skill-name>
```

不再需要手动查文档、装依赖，一键搞定！

### 4️⃣ Control UI 全新体验

- **Skills 状态筛选** - All / Ready / Needs Setup / Disabled
- **技能详情弹窗** - 需求、开关、API Key 配置、安装按钮
- **Agent 工作区文件预览** - 可展开的 Markdown 渲染

### 5️⃣ Discord 自动线程命名

新增 `autoThreadName: "generated"` 选项，让 LLM 自动生成简洁的线程标题！

---

## 🐛 重要修复

### 安全修复
- 关闭 `mediaUrl`/`fileUrl` 别名绕过漏洞
- 沙箱媒体派发策略对齐

### 稳定性提升
- Gateway 重启后自动唤醒中断的会话
- Docker 初始化循环问题修复
- 单个频道启动失败不再阻塞其他频道

### 平台适配
- **WhatsApp**: 群消息回声抑制、回复检测恢复
- **Telegram**: Forum Topic 路由恢复、照片上传容错
- **Discord**: 超时可见回复、网关崩溃修复
- **Slack**: DM 回复开销优化、运行时默认值收紧

---

## 📦 如何升级

```bash
# 全局安装用户
openclaw update

# 或使用 npm
npm update -g openclaw
```

> ⚠️ **Node.js 版本要求**: 最低 22.14+，推荐 24+
