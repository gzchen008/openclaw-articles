✨ ZeroClaw 的杀手锏

1. 可插拔架构

所有子系统都是 trait，换配置就行，不用改代码：

  • AI 模型：22+ providers，支持自定义 endpoint
  • 通信平台：Telegram/Discord/Slack/iMessage/Matrix/WhatsApp

2. 安全设计

  • 6 位配对码 + Bearer Token
  • 默认只绑定 localhost
  • 文件系统隔离（禁止访问 /etc、~/.ssh 等）
  • 命令白名单
  • Docker 沙箱可选

3. 从 OpenClaw 迁移

# 预览迁移（不修改文件）

# 正式迁移

⚡ 快速上手

# 安装
git clone https://github.com/zeroclaw-labs/zeroclaw.git
cd zeroclaw
cargo build --release
cargo install --path . --force

# 快速配置

# 开始聊天

# 启动网关
