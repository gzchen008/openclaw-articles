🚀 OpenClaw v2026.3.24 更新：Teams 集成升级、Control UI 全新体验、更多实用功能

OpenClaw 又一次带来了重磅更新！v2026.3.24 版本带来了Microsoft Teams 官方 SDK 迁移、Control UI 技能管理全面升级、Gateway API 兼容性扩展等众多新功能，同时修复了大量 bug，让你的 AI 助手体验更加流畅稳定。

💡 核心亮点

  • Microsoft Teams 官方 SDK 迁移 - 支持 AI Agent 最佳实践（流式回复、欢迎卡片、反馈机制、打字指示器等）
  • Control UI 技能管理全新改版 - 状态筛选、一键安装、详情对话框、Markdown 预览等
  • Gateway API 扩展 - 新增 /v1/models 和 /v1/embeddings 端点，兼容更多客户端
  • Slack 交互式回复增强 - 自动渲染按钮/选择器、优化默认设置
  • Docker 容器支持 - --container 参数让你直接在容器内执行命令

🆕 新功能详解

1. Microsoft Teams 全面升级 💬

这个版本完成了向官方 Teams SDK 的迁移，并实现了 AI Agent 的最佳用户体验：

  • 流式 1:1 回复 - 消息实时流式显示，不再等待完整响应
  • 欢迎卡片 - 新用户加入时显示带提示词启动器的欢迎卡片
  • 反馈/反思机制 - 用户可以对回复点赞/踩，帮助 AI 学习改进
  • 打字指示器 - AI 思考时显示"正在输入"状态
  • 原生 AI 标签 - 明确标识消息来自 AI 助手
  • 消息编辑/删除 - 支持编辑和删除已发送的消息（包括话题内消息）

2. Control UI 技能管理全新改版 🎛️

技能管理界面进行了全面升级，更加直观易用：

  • 状态筛选标签 - All / Ready / Needs Setup / Disabled 四个标签，一目了然
  • 一键安装 - 内置技能支持依赖项自动安装提示
  • 详情对话框 - 点击技能卡片查看详细信息、要求、配置等
  • API Key 配置 - 直接在界面中输入和保存 API Key
  • Agent 工作区文件预览 - 支持 Markdown 渲染、表格、代码块等富文本展示
  • 侧边栏 Markdown 样式 - 标题、列表、表格、代码块、引用等完整支持

3. Gateway API 兼容性扩展 🔌

新增 OpenAI 兼容端点，让更多客户端和 RAG 系统能够无缝接入：

  • /v1/models - 列出可用模型
  • /v1/embeddings - 支持向量嵌入
  • /v1/chat/completions 和 /v1/responses - 支持显式模型覆盖

4. 其他实用功能 ⚡

  • CLI 容器支持 - openclaw --container 或设置 OPENCLAW_CONTAINER 环境变量，直接在运行中的 Docker/Podman 容器内执行命令
  • Discord 自动线程命名 - 支持 autoThreadName: "generated"，用 LLM 生成简洁的线程标题
  • Slack 交互式回复 - 自动将简单的 Options: 行渲染为按钮/选择器
  • macOS App 配置界面 - 改用可折叠的树形侧边栏，更加清晰
  • Node 22 支持放宽 - 最低支持 Node 22.14+（仍推荐 Node 24）
  • CLI 更新预检 - 在执行 openclaw update 前检查 Node 版本兼容性，避免更新失败

🐛 重要修复

安全性修复 🔒

  • 沙箱媒体访问 - 修复 mediaUrl/fileUrl 别名绕过问题，确保工具和消息操作无法逃逸媒体根目录限制
  • 出站媒体/本地文件 - 对齐 workspaceOnly 配置，在非严格模式下保持发送主机本地文件和入站媒体的能力

稳定性修复 🛠️

  • Gateway 重启唤醒 - 重启后通过心跳唤醒中断的 Agent 会话，而不是仅发送通知；重试出站投递；保留显式的线程/话题路由
  • Docker 安装 - 修复全新 Docker 安装时的网络命名空间循环问题
  • 频道启动隔离 - 单个频道启动失败不再阻塞其他频道
  • WhatsApp 群组回显抑制 - 追踪最近发送的消息 ID，仅抑制匹配的群组回显，保留 owner 命令
  • Telegram 论坛话题 - 恢复 #General 话题路由（当 Telegram 省略论坛元数据时）
  • Discord 超时 - 入站 Discord worker 超时时发送可见的超时回复
  • Telegram 照片上传 - 预检照片尺寸和宽高比规则，无效时回退到文档发送

📥 如何更新

使用以下命令更新到最新版本：

openclaw update

或通过 npm 全局安装：

npm install -g openclaw

注意：此版本最低支持 Node 22.14+，推荐使用 Node 24。
