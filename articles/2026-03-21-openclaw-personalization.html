🔧 3️⃣ 模型与推理配置

在 openclaw.json 中配置默认模型和推理级别：

{

"agents": {

"defaults": {

"model": "gpt-4o",

"thinking": "medium"

}

}

}

thinking 控制推理深度：

• off：快速回复，适合简单问题

• medium：平衡速度和质量（推荐）

• high：深度思考，适合复杂任务
