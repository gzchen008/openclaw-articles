# GLM-5-Turbo：专为 Agent 设计的 AI 模型

## 首款 Agent 原生大模型

智谱 AI 发布 GLM-5-Turbo，这是业界首个**专为 Agent 场景原生设计**的大模型。

---

## 什么是 Agent 原生？

传统大模型是为"单次对话"设计的，而 GLM-5-Turbo 从底层就为 **Agent 工作流** 优化：

<p style="padding-left: 20px; margin: 4px 0;">• 🤖 理解任务链路：知道"为什么"做这件事</p>
<p style="padding-left: 20px; margin: 4px 0;">• 🔄 长上下文记忆：跨越多轮对话保持状态</p>
<p style="padding-left: 20px; margin: 4px 0;">• ⚡ 快速决策：减少推理延迟，适合实时调用</p>
<p style="padding-left: 20px; margin: 4px 0;">• 🎯 工具调用优化：结构化输出，精准匹配 API</p>

**一句话：GLM-5-Turbo 不是用来聊天的，是用来干活的。**

---

## 龙虾增强

GLM-5-Turbo 独有的"龙虾增强"能力（推理增强）：

### 📚 深度推理

不依赖外部工具链，模型内部就能完成复杂推理：
- 多步逻辑推导
- 约束条件分析
- 决策树遍历

### 🔄 自我反思

Agent 可以"思考自己的思考"：
- 评估推理过程
- 发现逻辑漏洞
- 主动纠正错误

### 🎯 目标对齐

始终记得最终目标：
- 短期任务服务于长期目标
- 避免陷入局部最优
- 保持方向一致性

## 如何在 OpenClaw 中使用

### 配置模型

```bash
openclaw config set model zai/glm-5-turbo
```

### 创建 Agent

```yaml
agent:
  model: zai/glm-5-turbo
  system: "你是一个专业的代码审查 Agent..."
```

### 优势

- **零配置**：开箱即用，无需调参
- **原生支持**：完美融入 OpenClaw Agent Loop
- **成本可控**：推理成本低，适合长时间运行
- **稳定可靠**：工具调用精准，减少错误重试
