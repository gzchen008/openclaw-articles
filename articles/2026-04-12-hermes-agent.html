🤔 Hermes Agent 到底是什么？

Hermes Agent 是一个自我进化的 AI Agent 框架，核心理念就一句话：The agent that grows with you——和你一起成长的 Agent。

它不像普通 Agent 只是"调用工具"，而是具备完整的学习闭环：

• 完成复杂任务后，自动创建可复用的 Skill

• 在使用过程中持续优化这些 Skill

• 定期提醒自己持久化重要知识到记忆系统

• 搜索历史对话，跨会话召回经验

• 通过 Honcho 辩证建模，越来越了解你是谁

简单说：你用得越多，它越懂你，越能干。

✨ 六大核心能力

1️⃣ 闭环学习系统（最核心的亮点）

这是 Hermes 区别于所有其他 Agent 框架的杀手锏。

普通 Agent 的记忆只是简单的键值存储。Hermes 的记忆是一个完整的认知系统：

• 自动 Skill 创建：完成一个复杂任务后，Hermes 会自动把解决过程提炼成可复用的 Skill

• Skill 自我优化：下次使用同一个 Skill 时，它会根据执行结果自动改进

• FTS5 全文搜索：可以搜索所有历史对话，用 LLM 摘要快速定位相关经验

• Honcho 用户建模：通过辩证式对话，逐步构建对你的深层理解

• 兼容 agentskills.io 开放标准：Skill 可以跨 Agent 平台共享

2️⃣ 多模型自由切换

不想被某个模型绑死？Hermes 完美支持：

• Nous Portal（自家模型服务）

• OpenRouter（200+ 模型一键切换）

• z.ai/GLM、Kimi/Moonshot、MiniMax

• OpenAI、Anthropic

• 自定义 Endpoint（接任何 OpenAI 兼容接口）

一条命令切换模型：hermes model，零代码改动，零锁定。

3️⃣ 全平台消息接入

Hermes 不只跑在终端。一个 Gateway 进程搞定所有平台：

• Telegram、Discord、Slack

• WhatsApp、Signal

• CLI（全功能 TUI 界面）

• 语音备忘录转录

• 跨平台对话连续性（Telegram 上聊到一半，换到 Discord 继续）

4️⃣ 随处运行，不仅限于笔记本

六种终端后端，想怎么跑就怎么跑：

• 本地直接运行

• Docker 容器化部署

• SSH 远程连接

• Daytona 无服务器（空闲时休眠，几乎不花钱）

• Singularity 高性能计算集群

• Modal 无服务器 GPU

5 美元的 VPS 就能跑，也可以跑在 GPU 集群上。你的 Agent 不应该被绑在你的笔记本上——在 Telegram 上跟它说话，它在云上干活。

5️⃣ 定时任务 & 子代理并行

定时任务：内置 Cron 调度器，自然语言描述任务，自动执行。每日报告、夜间备份、周度审计——全自动。

子代理并行：可以派生隔离的子代理，并行处理多个工作流。还支持 Python 脚本通过 RPC 调用工具，把多步骤管道压缩成零上下文消耗的单轮操作。

6️⃣ 从 OpenClaw 无缝迁移

如果你之前在用 OpenClaw，Hermes 提供了一键迁移：

• 自动检测 ~/.openclaw 目录

• 导入 SOUL.md（人格文件）、MEMORY.md、USER.md

• 迁移用户自建的 Skill

• 同步命令审批规则

• 转移消息平台配置和 API 密钥

• 支持 dry-run 预览模式

🚀 2分钟上手

安装只需一条命令：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

支持 Linux、macOS、WSL2，甚至 Android Termux。

安装完成后：

1️⃣ hermes — 启动交互式 CLI，开始聊天

2️⃣ hermes model — 选择你的 LLM 模型

3️⃣ hermes tools — 配置要启用的工具

4️⃣ hermes gateway — 启动消息网关（接入 Telegram 等）

5️⃣ hermes setup — 一键配置向导

💡 我的看法

Hermes Agent 最让我兴奋的不是某个具体功能，而是它的学习闭环设计理念。

大部分 Agent 框架还在卷"工具调用的数量"和"支持的平台数"，Hermes 已经在解决一个更根本的问题：Agent 如何从经验中学习，如何真正理解用户。

Honcho 的辩证用户建模 + FTS5 会话搜索 + 自动 Skill 创造，这三者组合起来，让 Agent 第一次有了类似人类的"经验积累"能力。

再加上灵活的部署方式（从 5 美元 VPS 到 GPU 集群）和多模型支持，Hermes 很可能成为 2026 年最值得关注的开源 Agent 框架之一。
