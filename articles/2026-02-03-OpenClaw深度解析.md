## 🎯 OpenClaw 是什么？

简单来说，**OpenClaw 是一个「AI 助手的操作系统」**。

它不是又一个聊天机器人，而是一个可以让你把各种 AI 模型（GPT、Claude、Kimi 等）接入到各种通讯平台（微信、Discord、Telegram、Slack 等）的桥梁。

### 核心能力一览

| 能力 | 说明 | 场景举例 |
|------|------|----------|
| 🌉 **多平台接入** | 一次配置，同时管理多个平台 | Discord 群助手 + 微信客服 + Telegram Bot |
| 🤖 **多模型切换** | GPT-4、Claude、Kimi 自由切换 | 写代码用 Claude，写文案用 GPT |
| 🧠 **长期记忆** | 自动记录对话历史，支持语义搜索 | "上周我让你整理的资料在哪？" |
| ⏰ **定时任务** | 设置一次，每天自动执行 | 每天早上 9 点自动推送行业资讯 |
| 🛠️ **扩展技能** | 安装各种 Skill 扩展功能 | 查天气、搜网页、控制浏览器 |
| 🔒 **私有化部署** | 数据存在本地，完全可控 | 敏感信息不出境 |

## 🛠️ 技术架构浅析

OpenClaw 采用**模块化设计**，主要包含：

```
┌─────────────────────────────────────────┐
│           通讯平台层 (Channels)          │
│  Discord │ Telegram │ Slack │ WhatsApp  │
└────────────┬────────────────────────────┘
             │
┌────────────▼────────────────────────────┐
│           网关层 (Gateway)               │
│     统一消息格式 │ 身份验证 │ 路由分发     │
└────────────┬────────────────────────────┘
             │
┌────────────▼────────────────────────────┐
│           代理层 (Agents)                │
│   会话管理 │ 记忆搜索 │ 工具调用 │ 技能   │
└────────────┬────────────────────────────┘
             │
┌────────────▼────────────────────────────┐
│           模型层 (Models)                │
│   OpenAI │ Anthropic │ Gemini │ 本地模型 │
└─────────────────────────────────────────┘
```

**核心特点：**
- **Gateway（网关）**：负责与各平台对接，统一消息格式
- **Agent（代理）**：处理对话逻辑，管理上下文和记忆
- **Skill（技能）**：可插拔的功能扩展，按需安装
- **Cron（定时任务）**：后台自动执行预设任务

这种设计的好处是**松耦合**——你可以只使用其中某一层，也可以整体使用，非常灵活。

如果你看到这里觉得感兴趣， here's your action plan：

- 一台能 24 小时开机的电脑（Mac/Linux/Windows 都可以）

### 第二步：安装 OpenClaw
```bash
# macOS / Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows
# 下载安装包后双击安装
```

### 第三步：配置你的第一个平台
推荐从 **Discord** 开始（配置最简单）：
1. 创建 Discord Bot，获取 Token
2. 配置 `openclaw.json` 文件
3. 运行 `openclaw start`

- 安装需要的 Skill（天气、网页搜索等）
