🚀 3分钟安装教程

步骤1：安装 OpenClaw

Linux / MacOS：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows：

iwr -useb https://openclaw.ai/install.ps1 | iex

安装成功后，打开 OpenClaw Dashboard，看到管理后台就代表安装成功 ✅

步骤2：安装飞书插件

在终端中执行：

npx -y @larksuite/openclaw-lark-tools install

执行过程中，你可以选择：

• 新建机器人（扫码一键创建）

• 关联已有机器人（输入 App ID 和 App Secret）

步骤3：快速授权

创建完成后，在飞书对话中发送：

/feishu auth

这会一次性完成所有用户授权，包括：

• 文档读写权限

• 消息读取权限

• 日历权限

• 任务权限

步骤4：验证安装

在飞书对话中发送：

/feishu start

如果返回了版本号信息，就代表安装成功！🎉

⚙️ 常用配置命令

开启流式输出（效果更流畅）

openclaw config set channels.feishu.streaming true

开启多任务并行

让 AI 在话题群中，针对每个话题独立上下文、并行执行：

openclaw config set channels.feishu.threadSession true

诊断命令

在飞书对话中发送：

• /feishu start - 确认是否安装成功

• /feishu doctor - 检查配置是否正常

• /feishu auth - 批量完成用户授权

🔧 常见问题

Q1：安装时报错 "cannot find module xxx"

原因：系统没有安装插件的依赖

解决方法：进入插件安装目录，运行 npm install

Q2：权限不足怎么办？

解决方法：

1️⃣ 在飞书开放平台 → 你的应用 → 权限管理

2️⃣ 点击"批量导入/导出权限"

3️⃣ 导入所需权限（文档中有完整列表）

4️⃣ 点击"申请开通"

Q3：升级到新版本后工具无法调用

解决方法：

在 openclaw.json 中添加：

{
  "tools": {
    "profile": "full",
    "sessions": {
      "visibility": "all"
    }
  }
}
