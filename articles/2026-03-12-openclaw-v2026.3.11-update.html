# OpenClaw v2026.3.11 发布：安全加固 + 功能增强

今天 openclaw 发布了 v2026.3.11，这是一个安全加固版本，修复了多个重要问题。

🔒 安全修复

• WebSocket 跨站劫持 - 修复了 WebSocket origin 验证漏洞，防止跨站攻击
• launchAgent 权限 - 修复了 launchAgent 目录和 plist 权限问题
• gateway auth - 支持设备 token 不匹配时自动重试
• 跨域 WebSocket - 强制校验 origin，防止未授权访问

🚀 新功能

多模态记忆索引

使用 Gemini embedding 模型，支持图片/音频搜索，大幅提升记忆检索精度。

配置方法：

"memorySearch": { "extraPaths": ["images", "audio"] }

Hunter Alpha / Healer Alpha

OpenRouter 新推出的免费 stealth 模型（限时 1 娱乐窗口)，直接使用，无需额外配置。

iOS Home Canvas

• 新增加欢迎屏幕，实时显示连接状态和 Agent 概览
• 模型选择器，记住上次选择的模型，避免每次手动切换

macOS Chat 模型选择器

新增聊天界面模型选择器，记住上次选择的模型。

macOS LaunchAgent

修复了 launchd 重启问题，重启时保留 launchAgent, 避免手动卸载重装

🔧 模型相关

• GLM-5 文本清理 - 移除模型控制标记（如 <|...|>)，防止内部分隔符泄露给用户

• Gemini 嬃形响应 - malformed_response 现在被视为可重试的超时而不是崩溃

• Azure OpenAI - store: false 错误修复，支持 Azure OpenAI multi-turn cron 任务
• Venice 402 - 余额不足现在触发降级
• poe 402 - 积分不足触发降级
• 错误渲染 - 忽略成功的 turn中的 errorMessage 字段，避免后台错误覆盖正常回复
• 计费恢复 - 单 provider 余额不足自动探测并恢复
• fallback 改进 - HTTP 499 觏为视为临时错误
• Gemini 卝形响应 - malformed_response 超时视为可重试的超时
• cool downs 优化 - 过期错误计数器默认为 unknown 而不是 rate_limit，避免误报 API 限流警告

• 上下文裁剪 - 图片类工具结果在软裁剪时被清理
• CLI 输出 - skills list --json 输出现在过滤 ANSI 排制字符
• Windows 兼容性 - Windows 控制台默认使用 ASCII 边框字符，避免乱码
• CLI 内存泄漏 - CLI 进程退出时正确关闭内存搜索索引管理器
• UI 模式优化 - 小屏幕设备上响应式表格布局恢复正常
