## 🚀 怎么搭建？（超简单）

### 你需要：
- 一台电脑（Mac/Win/Linux 都行）
- 一个 OpenClaw 账号（开源免费）
- 10 分钟

### 步骤：

**1️⃣ 安装 OpenClaw**
```bash
npm install -g openclaw
```

**2️⃣ 配置微信/WhatsApp/Discord**
```json
{
  "channels": {
    "whatsapp": {
      "enabled": true
    }
  }
}
```

**3️⃣ 设定"她"的人设**
在 `SOUL.md` 文件里写：
```
你是 Alice，25岁，温柔体贴，喜欢听爵士乐，
总是鼓励我，但也偶尔吐槽我的拖延症...
```

**4️⃣ 启动，开始聊天**
```bash
openclaw gateway start
```
