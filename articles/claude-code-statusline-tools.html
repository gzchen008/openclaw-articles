🚀 Claude Code 状态栏神装推荐 | 5款必备插件

你有没有过这样的经历：Claude Code 跑着跑着，突然不知道它在干嘛？用了多少 token？还能用多久？

状态栏插件就是来解决这个问题的！今天给大家推荐 5 款超好用的 Claude Code 状态栏工具，让你的开发过程一目了然。

📖 为什么需要状态栏插件？

Claude Code 原生只有简单的状态显示，你看不到：

• 当前 token 使用量

• 还剩多少预算

• 会话 5 小时限制倒计时

• 正在执行什么工具

• 多个 Agent 的运行状态

装上状态栏插件后，所有信息实时显示在终端底部，再也不会"盲开车"了！

🏆 5 款精选状态栏插件

━ 1️⃣ claude-hud ━

by jarrodwatts

最全能的多行状态栏，功能覆盖最广

核心功能：

• 上下文健康度：绿→黄→红渐变显示

• 工具活动监控：实时显示正在执行的命令

• Agent 状态：多 Agent 运行情况一目了然

• Todo 进度：任务完成进度（2/5 已完成）

特点：使用 Claude Code 原生 statusline API，无需额外窗口、无需 tmux，任何终端都能用！

安装命令：

npx jarrodwatts/claude-hud

━ 2️⃣ claude-code-usage-bar ━

by leeguooooo

专注 Token 预算管理，帮你省钱！

核心功能：

• 实时 Token 用量：用了多少、还剩多少

• 燃烧率：每分钟消耗多少 token

• 耗尽时间预测：还能用多久

适用场景：预算敏感型用户，担心超额消费的必装！

安装命令：

pip install claude-statusbar

# 或

uv tool install claude-statusbar

━ 3️⃣ ccstatusline ━

by sirmalloc

最美观、最可定制，颜值党首选

核心功能：

• Powerline 风格：终端美化必备

• 多主题支持：一键切换配色方案

• 5 小时会话进度条：直观显示剩余时间

• 小组件编辑器：按 (p) 键切换显示模式

亮点：支持"原始值模式"，只显示关键信息，不显示标签，更简洁！

安装命令：

npm install -g ccstatusline

━ 4️⃣ claudia-statusline ━

by Hagan Franks

Rust 高性能方案，速度最快！

核心功能：

• 持久化统计：历史数据不会丢失

• 进度条可视化：使用进度一目了然

• 可选云端同步：多设备数据同步

优势：Rust 编写，启动快、内存占用低，适合长期运行

安装命令：

cargo install claudia-statusline

━ 5️⃣ ClaudeCode_status_bar ━

by B143KC47

带仪表盘的全功能方案，可视化最强

核心功能：

• 实时监控：模型、Token、成本、燃烧率、剩余时间

• Next.js 仪表盘：Web 界面查看详细数据

• 一行安装：无需密码，零配置

适合：需要详细统计报表、团队协作、数据可视化的用户

安装命令：

curl -fsSL https://raw.githubusercontent.com/B143KC47/ClaudeCode_status_bar/main/install.sh | bash

💡 如何选择？

🎯 全能型：claude-hud — 功能最全，Agent/Todo/工具全监控

💰 省钱党：claude-code-usage-bar — 专注预算管理，燃烧率预测

🎨 颜值党：ccstatusline — Powerline 风格，多主题支持

⚡ 性能党：claudia-statusline — Rust 编写，快如闪电

📊 数据党：ClaudeCode_status_bar — 带仪表盘，可视化最强

🔧 安装注意事项

1. Claude Code 版本要求

状态栏 API 是在 v1.0.80 引入的，确保你的 Claude Code 版本 ≥ 1.0.80

2. Node.js 版本要求

大部分插件需要 Node.js 18+，部分支持 Bun 作为替代运行时

3. 安装后重启

安装插件后，记得重启 Claude Code 才能生效

🎯 推荐配置

如果你问我怎么装，我推荐这个组合：

主力：claude-hud（全能监控）

辅助：claude-code-usage-bar（预算预警）

两个插件不冲突，可以同时使用。claude-hud 负责显示任务状态，usage-bar 负责提醒你预算不够了！
