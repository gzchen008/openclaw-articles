━ ⚙️ 高级配置 ━

自定义浏览器行为

如果你想更精细地控制浏览器，可以在配置中添加：

{

"browser": {

"enabled": true,

"type": "chrome",

"headless": false,

"timeout": 30000,

"viewport": {

"width": 1920,

"height": 1080

}

}

}

参数说明：

• headless：是否无头模式（不显示浏览器窗口）

• timeout：操作超时时间（毫秒）

• viewport：浏览器窗口大小

添加浏览器扩展

如果你需要使用浏览器扩展（如广告拦截器），可以配置扩展路径：

{

"browser": {

"extensions": [

"/path/to/extension1",

"/path/to/extension2"

]

}

}

━ ⚠️ 常见问题 ━

Q: 浏览器打开后没有反应？

A: 检查网络连接和网站是否可访问。也可能是超时设置太短，尝试增加 timeout 值。

Q: 有些网站无法访问？

A: 某些网站有反爬虫机制，可能需要设置 User-Agent 或使用代理。在配置中添加：

{

"browser": {

"userAgent": "Mozilla/5.0 ...",

"proxy": "http://proxy-server:port"

}

}

Q: 浏览器占用太多内存？

A: 启用无头模式（headless: true），可以大幅降低内存占用。

Q: 如何处理需要登录的网站？

A: 可以配置浏览器使用已有的用户数据目录，这样能保持登录状态：

{

"browser": {

"userDataDir": "/path/to/chrome/profile"

}

}

━ 🎯 最佳实践 ━

1. 设置合理的超时时间

不同网站的加载速度不同，建议根据实际情况设置超时时间：

• 国内网站：10-15 秒

• 国外网站：20-30 秒

• 复杂页面：30-60 秒

2. 使用无头模式节省资源

如果是定时任务，建议使用无头模式，不占用屏幕空间，内存占用也更低。
