# Dokobot: 隐私优先的浏览器抓取工具

## 隐私是一切的核心

你的浏览数据始终由你掌控。本地模式将一切处理留在你的设备上。远程模式支持端到端加密，即使服务器也无法读取你的内容。

- 本地优先，无需登录
- 端到端加密（scrypt + AES-256-GCM）
- 内容绝不存储，会话数据即用即销

## 基于像素的视觉提取

Dokobot 引擎在像素级别分析渲染后的页面，而非 DOM 树。它从视觉坐标检测内容边界，不受 HTML/CSS 改版影响。

- 基于视觉，而非 DOM
- 无需站点适配器，任意网页通用
- 不受网站改版影响

## 自动滚动续读无忧

Dokobot 自动滚动浏览整个长页面，逐段捕获包括懒加载在内的所有内容。

- 智能自动滚动，自动检测懒加载
- 会话级续读，暂停后精确恢复
- 自适应检测，无新内容提前停止

## 任意网页 LLM 直接可用

Dokobot 输出 LLM 一目了然的结构化文本，Token 减少 99%。

- 按栏分组，保留页面实际布局
- 图片与链接引用
- Token 减少 99%

## 本地免费无限抓取

通过自己的浏览器免费无限抓取网页，无需服务器。Pro 提供远程浏览器和无限并发连接，仅 $9.90/月。

## 你的浏览器随处可达

AI Agent 通过一个 API 即可抓取你真实浏览器中的页面。无需代理，无需云浏览器，无需共享账密。

## Agent 的最佳搭档

一个 Skill，让你的 Agent 用 doko read 提取任意网页，用 doko search 搜索全网信息。
