## 🎨 Canva 基础设置

### 1. 创建设计
1. 打开 canva.com（免费版即可）
2. 点击「创建设计」→「自定义大小」
3. 输入尺寸：**900 x 500 像素**（正文配图）或 **900 x 383 像素**（封面）

### 2. 配色方案（已保存）
```
主色深蓝：#1e3a5f（标题、重点）
强调金色：#f5a623（数据、高亮）
辅助灰色：#6b7280（说明文字）
背景白色：#ffffff
成功绿色：#10b981（上涨、正面）
警示红色：#ef4444（下跌、风险）
```

### 3. 字体推荐
- **标题**：思源黑体 Bold / 阿里巴巴普惠体 Bold
- **正文**：思源黑体 Regular / 苹方 Regular
- **数字**：DIN / Roboto Mono（等宽，显示数据更专业）

### 【配图3】战绩对比曲线图（900 x 500px）

**模板路径**：Canva → 搜索「line chart」→ 选择双曲线模板

**制作步骤**：
1. **背景**：白色
2. **添加图表**：使用Canva的「图表」工具 → 折线图
   - 蓝色曲线：AI系统（标记几个高点，整体向上）
   - 红色曲线：沪深300（波动大，整体平缓或向下）
3. **数据标注**：
   - 蓝色终点：「+18.5%」金色标签
   - 红色终点：「-1.5%」灰色标签
   - 中间加：「跑赢20%」金色大标签
4. **图例**：底部标注两条线分别代表什么

**数据点参考**（从左到右）：
- AI系统：0%, 3%, 5%, 8%, 12%, 15%, 18.5%
- 沪深300：0%, -2%, 1%, -3%, -1%, -2%, -1.5%

## 💡 Canva 制作技巧

### 快捷键
- `Ctrl + D`：复制元素
- `Ctrl + G`：组合元素
- `Alt + 拖动`：复制并移动
- `Shift + 拖动`：保持比例缩放

### 素材搜索关键词
| 需要 | 搜索词 |
|------|--------|
| 金融图标 | finance, stock, trading |
| 机器人/AI | robot, AI, artificial intelligence |
| 图表 | chart, graph, data |
| 箭头 | arrow, direction |
| 手机界面 | mobile app, phone mockup |
| 背景 | gradient, technology background |

### 免费素材来源
1. Canva自带素材库（筛选「免费」）
2. unDraw.co（免费插画，可改色）
3. icons8.com（免费图标）
4. flaticon.com（免费矢量图标）
