## 🚀 安装步骤（3 种方式）

### 方式一：Docker 安装（推荐，5分钟搞定）

**前置要求：**
- 安装了 Docker 和 Docker Compose
- 有一点终端使用基础

**1️⃣ 创建项目目录**

```bash
mkdir -p ~/openclaw && cd ~/openclaw
```

**2️⃣ 创建 docker-compose.yml**

```yaml
version: '3.8'
services:
  openclaw:
    image: openclaw/openclaw:latest
    container_name: openclaw
    ports:
      - "3000:3000"
    volumes:
      - ./data:/app/data
      - ./memory:/app/memory
    environment:
      - OPENCLAW_API_KEY=YOUR_API_KEY_HERE
      - DEFAULT_MODEL=gpt-4o-mini
    restart: unless-stopped
```

💡 **说明：**
- `YOUR_API_KEY_HERE` 换成你的 OpenAI API Key（或其他模型提供商的 Key）
- 端口 3000 可以自定义
- 数据会持久化到本地的 ./data 和 ./memory 目录

**3️⃣ 启动服务**

```bash
docker-compose up -d
```

看到 `started` 就表示成功了！

**4️⃣ 验证安装**

浏览器打开 `http://localhost:3000/status`，看到状态页面就说明跑起来了。

---

### 方式二：Node.js 安装（适合开发者）

**前置要求：**
- Node.js 18+
- npm 或 yarn

**1️⃣ 全局安装**

```bash
npm install -g openclaw
```

**2️⃣ 初始化配置**

```bash
openclaw init
```

会生成默认配置文件 `openclaw.config.json`。

**3️⃣ 编辑配置**

```bash
openclaw config edit
```

至少需要配置：
- API Key（OpenAI / Claude / Gemini 等）
- 要启用的平台（Discord、Telegram 等）

**4️⃣ 启动服务**

```bash
openclaw start
```

## ⚙️ 基础配置详解

安装完成后，需要配置几个关键参数：

### 1. AI 模型配置

在配置文件中设置：

```json
{
  "models": {
    "default": "gpt-4o-mini",
    "providers": {
      "openai": {
        "apiKey": "YOUR_OPENAI_API_KEY"
      },
      "anthropic": {
        "apiKey": "YOUR_ANTHROPIC_API_KEY"
      }
    }
  }
}
```

**推荐模型选择：**
- **GPT-4o-mini** - 性价比高，日常对话足够
- **Claude 3.5 Sonnet** - 代码能力强，逻辑性好
- **Gemini 1.5 Flash** - Google 出品，速度快

### 2. 平台接入配置

**Discord 配置示例：**

```json
{
  "channels": {
    "discord": {
      "enabled": true,
      "botToken": "YOUR_DISCORD_BOT_TOKEN",
      "allowedGuilds": ["YOUR_GUILD_ID"]
    }
  }
}
```

💡 **获取 Bot Token 步骤：**
1. 访问 <https://discord.com/developers/applications>
2. 创建 New Application
3. 进入 Bot 页面，点击 "Reset Token"
4. 复制 Token 填入配置

**Telegram 配置示例：**

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "YOUR_TELEGRAM_BOT_TOKEN"
    }
  }
}
```

💡 **获取 Telegram Bot Token：**
1. 在 Telegram 搜索 @BotFather
2. 发送 `/newbot`
3. 按提示设置名称和用户名
4. 复制返回的 Token

## 🛠️ 常见问题排雷

### ❌ 问题 1：启动后无法访问

**可能原因：**
- 端口被占用
- 防火墙阻止

**解决方案：**
```bash
# 检查端口占用
lsof -i :3000

# 更换端口（docker-compose.yml 中修改）
ports:
  - "8080:3000"
```

### ❌ 问题 2：API Key 报错

**错误提示：** `Invalid API key` 或 `Authentication failed`

**排查步骤：**
1. 确认 Key 是否复制完整（不要有多余空格）
2. 确认 Key 对应的模型服务商是否正确
3. 检查 Key 是否有余额/额度

### ❌ 问题 3：Discord 机器人不响应

**可能原因：**
- Bot Token 错误
- 没有给 Bot 发送消息的权限

**解决方案：**
1. 重新生成 Token
2. 在 Discord Developer Portal 的 OAuth2 URL Generator 中勾选：
   - `bot`
   - `Send Messages`
   - `Read Message History`
