🚀 OpenClaw v2026.3.28 更新发布

版本 v2026.3.28

发布时间：2026年3月29日

📊 发布概览

🎯 本次更新亮点：

  • 🔥 xAI 集成增强：新增 Grok 插件和 x_search 功能
  • 🎨 MiniMax 图像生成支持：支持图像生成和编辑
  • 🛡️ 插件审批机制：支持工具调用前的用户审批
  • 🔗 ACP 频道绑定：支持 Discord、BlueBubbles、iMessage 绑定
  • 📱 多个平台修复：Telegram、WhatsApp、Matrix 等平台改进

🚨 重要变更

注意：以下变更可能影响现有配置

移除了已弃用的 Qwen `qwen-portal-auth` OAuth 集成，需要迁移到 Model Studio API Key 认证
Config/Doctor 停止自动配置超过两个月的旧配置迁移，旧密钥现在会失败验证

✨ 新功能

🤖 xAI 集成增强

将绑定的 xAI 提供者移到 Responses API，新增 `x_search` 功能
在 `openclaw onboard` 期间自动启用 xAI 插件，支持 Grok 网络搜索

🎨 MiniMax 图像生成

新增 `image-01` 模型图像生成支持
支持图像到图像编辑，具有宽高比控制

🔧 插件审批机制

插件现在可以在工具调用前暂停执行并请求用户审批
支持多种审批方式： exec 审批叠加层、Telegram 按钮、Discord 交互等

🔗 ACP 频道绑定

为 Discord、BlueBubbles 和 iMessage 添加当前对话 ACP 绑定
支持 `/acp spawn codex --bind here` 将当前聊天转为 Codex 工作空间

🔧 功能改进

OpenAI `apply_patch` 默认启用，权限对齐改进
插件 CLI 后端支持增强，新增 Gemini CLI 后端
Podman 容器设置简化，支持用户级安装
Slack 工具操作新增 `upload-file` 支持
文件传输标准化，支持 Microsoft Teams 和 Google Chat
Matrix TTS 改为原生语音气泡格式
新增 `openclaw config schema` 命令
配置 TTS 自动迁移旧语音配置
内存插件架构改进
MiniMax 模型目录优化，仅保留 M2.7
插件运行时增强，支持 `runHeartbeatOnce`
代理压缩改进，使用率检查增强
