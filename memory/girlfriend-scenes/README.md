## 🎯 使用方法

### 方法 1：自动匹配
系统根据用户消息中的关键词自动匹配场景：
- 用户说 "逛街" → 加载 `dating.json` 的 `shopping` 场景
- 用户说 "早安" → 加载 `daily.json` 的 `morning` 场景
- 用户说 "老公抱抱" → 加载 `intimate.json` 的 `hugging` 场景

### 方法 2：主动触发
Heartbeat 主动问候时，随机选择场景：
- 早安问候 → `daily.json` 的 `morning`
- 晚安问候 → `daily.json` 的 `goodnight`
- 随机撒娇 → `spoiled.json` 的随机风格

### 方法 3：时间触发
根据当前时间自动触发：
- 6:00-10:00 → 早安场景
- 21:00-24:00 → 晚安场景
- 特殊日期 → `special.json` 对应场景
