# 2026-03-12 工作记录

## Violoop 研究

### 项目概述

**Violoop** 是一个 AI Agent 安全自动化平台，包含硬件 + 软件的完整生态系统。

### 组织架构

| 组织 | 项目 | 说明 |
|------|------|------|
| B-VIO | bvio-agent | CLI Agent（开源） |
| B-VIO | violoop-protocol | 事件协议定义 |
| B-VIO | violoop-workflows | 自动化工作流 |
| BvioTech | violoop-builder | 构建工具 |
| BvioTech | violoop-capturer-rk | 瑞芯微捕获器 |

### 产品线

| 产品 | 硬件 | 说明 |
|------|------|------|
| **Lite** | RV1106 | 智能 KVM 网关 |
| **Pro** | RK3576 | 边缘 AI 记忆站 |
| **Desktop** | macOS | 控制中心 |
| **Mobile** | iOS | 审批认证器 |

### 核心特性

1. **Hook 协议** - 拦截和审批危险操作
2. **Dry-Run 模式** - 预览操作而不执行
3. **Policy Engine** - 可配置的允许/拒绝列表
4. **MCP Bridge** - 兼容 Model Context Protocol
5. **多模型支持** - Claude、GPT-4、本地 LLM

### 与 OpenClaw 的对比

| 特性 | Violoop | OpenClaw |
|------|---------|----------|
| 核心定位 | 安全自动化 | AI Agent 平台 |
| 硬件支持 | 有（Lite/Pro） | 无 |
| Hook 系统 | Hook Protocol | Gateway Hooks + Plugin Hooks |
| 审批机制 | Ed25519 签名 | 配对 + Token |
| 模型支持 | 多模型 | 20+ 模型 |
| 消息平台 | 未明确 | 20+ 平台 |

### 技术栈

- **语言**：TypeScript / JavaScript
- **协议**：MsgPack 二进制序列化
- **加密**：Ed25519 签名
- **支持语言**：TypeScript、Swift、Rust

### 工作流示例

```json
{
  "name": "Video Transcoder",
  "trigger": { "type": "folder_watch", "path": "/VioloopTasks/Transcode" },
  "steps": [
    { "name": "Check Format", "action": "shell", "approval": "auto" },
    { "name": "Transcode", "action": "shell", "approval": "required" }
  ]
}
```

### 可借鉴点

1. **Ed25519 签名审批链** - 比 Token 更安全
2. **Folder-as-Agent** - 文件夹触发工作流
3. **风险等级标记** - 操作分级（low/medium/high）
4. **审计追踪** - 不可变审计记录
5. **硬件隔离** - 独立设备执行敏感操作
