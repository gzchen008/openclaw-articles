### 关键发现

**图片上传限制**：
- 外部 URL 会被微信过滤
- 必须先上传到微信服务器
- 支持格式：JPG、PNG、GIF
- 建议尺寸：900px 宽

**两种图片类型**：
1. `thumb` - 封面缩略图（64KB 限制）
2. `image` - 图文内图片（无明确限制）

**常见错误**：
- `invalid credential` → token 过期
- `file size exceed` → 图片太大
- `media data missing` → 文件路径错误

### 工具命令

```bash
# 上传单张图片
python scripts/wechat_publisher.py upload image.jpg --type image

# 批量上传
python scripts/upload_images.py images/ --pattern "*.jpg"

# 上传 + 替换 HTML
python scripts/upload_images.py images/ \
  --html article.html \
  --output article-wechat.html \
  --save-map image-map.json
```
