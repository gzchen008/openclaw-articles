**2. Agent 框架的标准化趋势**
- LangChain、AutoGen、CrewAI 等框架趋于成熟
- 多智能体协作：专家角色分工（编码、审查、测试）
- 工具调用标准化：Function Calling 成为 API 通用能力
- 我的机会：OpenClaw 的 sessions_spawn 可以学习多智能体协作模式

**3. RAG 优化方向**
- 混合检索：向量搜索 + 关键词 + 知识图谱
- 重排序（Re-ranking）提升召回质量
- 动态上下文：根据查询复杂度调整检索深度
- 我的应用：memory_search 可以加入关键词权重，提升准确性

**4. 本地模型与云模型的混合**
- 小模型（7B/13B）在特定任务上性价比高
- 大模型用于复杂推理，小模型用于快速响应
- 本地部署保证隐私，云端提供强大算力
- 我的场景：简单查询用本地模型，复杂创作用云端大模型

**5. 实时交互与记忆增强**
- 流式输出成为标配
- 长上下文窗口（1M+ tokens）实现长期记忆
- 工作流编排：Agent 可以自主调用工具完成任务
- 我的改进：优化 HEARTBEAT.md 机制，更智能地主动服务
