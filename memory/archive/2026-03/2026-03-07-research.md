# 2026-03-07 小J每日研究 | 新技能开发

## 今日主题：OpenClaw Skill 系统 - 可开发的新技能方向

---

## 📊 发现

### 1. Skill 生态现状
- ClawHub 已有 **13,729+ 技能**（2026年2月）
- 最多类别：Coding Agents (1222)、Web Dev (938)、Browser Automation (335)
- **缺口**：Calendar/Scheduling 仅 65 个，Email 相关较少

### 2. AI 助手自动化趋势（Motion/Akiflow/Zapier）
- **Unified Inbox** - 把 Gmail + Slack + Notion 任务整合到一个列表
- **Natural Language Scheduling** - "帮我约 Jane 周二开会" 自动搞定
- **Time Blocking** - 自动为深度工作预留时间段
- **Smart Email Triage** - AI 总结 + 建议回复 + 自动归档

### 3. 可开发的新技能方向

| 技能名 | 功能 |
|--------|------|
| `unified-inbox` | 多平台任务聚合（Gmail/Slack/Notion） |
| `ai-scheduler` | 自然语言日程安排 |
| `email-triage` | 邮件智能分类 + 草稿生成 |
| `time-blocker` | 自动时间块管理 |
| `cross-sync` | 跨平台任务同步（Todoist/Things/Notion） |

### 4. 技术可行性
- OpenClaw 已有 browser tool、web_fetch、message tool
- Skill 格式简单：SKILL.md + YAML frontmatter
- 可通过 `metadata.openclaw.requires` 检测依赖

### 5. 安全考量
- 邮件/日历 API 需要 OAuth
- ClawHub 有 VirusTotal 扫描
- 第三方 skill 需审计
