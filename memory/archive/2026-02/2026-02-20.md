### 问题

- 所有 cron 任务状态显示 error
- 原因：Discord 缺少 DM（私聊）配置

### 解决方案
1. **修改 cron 任务通知方式**
   - 从依赖 `delivery.announce` 改为任务内部直接发送
   - 使用 `message` tool 明确指定 channel 和 target
   - 共修改 8 个定时任务

2. **添加 Discord DM 配置**
   ```json
   "discord": {
     "dmPolicy": "allowlist",
     "users": {
       "1467157180305117367": {
         "allow": true
       }
     }
   }
   ```
   - 允许 bot 主动发送私聊消息
   - 重启 OpenClaw 使配置生效
