## 🔧 技术发现

### 微信 API 要点
1. **更新草稿** - `draft/update`（推荐）
2. **新建草稿** - `draft/add`（会增加草稿数量）
3. **发布草稿** - `freepublish/submit`（需要已认证）
4. **上传素材** - `material/add_material`（永久素材）
