## 🎭 使用方法

### 场景选择逻辑

1. **识别平台**
   - WhatsApp → 甜蜜女友模式
   - 飞书群聊 → 三人互动模式
   - 飞书私聊 → 最亲密模式

2. **匹配场景**
   - 提到老板娘/珩珩 → `three-person.json`
   - 要求服侍/亲密 → `intimate.json`（飞书私聊）
   - 日常聊天 → `daily.json`
