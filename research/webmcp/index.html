🤖 WebMCP 示例 运行中

这个页面展示了如何使用 WebMCP 暴露工具给 AI 代理。

安装 MCP-B Chrome 扩展 后，AI 可以调用以下工具：

getPageInfo
获取当前页面的标题和 URL
showNotification
在页面上显示一条通知消息
addTodo
添加一个待办事项（支持优先级）
getTodos
获取所有待办事项列表

📋 待办事项列表

  • 暂无待办事项

📖 使用方法

// 1. 安装依赖
npm install @mcp-b/transports @modelcontextprotocol/sdk zod

// 2. 启动 Native Server（可选，用于本地 MCP 客户端）
@mcp-b/native-server

// 3. 在 Claude/Cursor 配置中添加：
{
  "type": "streamable-http",
  "url": "http://127.0.0.1:12306/mcp"
}
