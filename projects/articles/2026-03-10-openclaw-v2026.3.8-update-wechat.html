OpenClaw 2026.3.8 发布：备份功能、智能搜索增强、多项体验优化

OpenClaw 又一次迎来重要更新！v2026.3.8 带来了全新的本地备份系统、Brave 智能搜索增强、Talk 模式静默检测配置，以及大量 Bug 修复和体验优化。

🆕 重要新功能

📦 CLI 备份系统

全新的备份命令让你的 OpenClaw 配置更安全：

openclaw backup create

openclaw backup create --only-config

openclaw backup verify

备份功能支持：

• 本地状态归档

• 配置文件独立备份

• 备份清单和负载验证

• 在执行破坏性操作前提供备份引导

🔍 Brave 智能搜索增强

新增 LLM Context 模式，让搜索结果更智能：

tools:

web:

search:

brave:

mode: "llm-context"

启用后，web_search 将调用 Brave 的 LLM Context 端点，返回带有来源元数据的提取式摘要片段，让 AI 获取更精准的搜索结果。

🎙️ Talk 模式静默检测

为语音交互新增可配置的静默超时：

talk:

silenceTimeoutMs: 2000 # 静默 2 秒后自动发送

不同平台保留各自的默认暂停窗口，现在你可以自定义等待时间。

🖥️ TUI 智能识别

TUI 现在能从当前工作区自动推断活动 Agent，无需手动指定。同时保留显式 agent: 会话目标的优先级。

📋 ACP 来源追踪

新增 ACP 入口来源元数据和可见收据注入：

openclaw acp --provenance off|meta|meta+receipt

让 OpenClaw Agent 能够保留和报告 ACP 来源上下文，并附带会话追踪 ID。

🔧 版本信息增强

openclaw --version 现在会显示 Git commit hash（当元数据可用时），方便调试和问题追踪。

🐛 重要修复

macOS 体验优化

• LaunchAgent 恢复：更新时自动重新启用被禁用的 LaunchAgent 服务

• App/Chat UI：浏览器代理路由优化、纯文本粘贴语义保留、权限状态刷新改进

• Onboarding：新增远程 Gateway Token 字段，支持远程模式配置

通讯平台修复

• Telegram DM：修复同一 DM 可能触发重复回复的问题

• Matrix DM：改进对异常 homeserver 的容错处理

• Feishu 插件：修复安装后缓存导致的重复下载提示

其他修复

• Android Play：精简权限，移除自更新、后台定位等不必要的能力

• 插件加载：优先使用内置版本，避免被 npm 安装的同名插件覆盖

• 浏览器中继：新增 browser.relayBindHost 配置，支持 WSL2 等跨命名空间场景
