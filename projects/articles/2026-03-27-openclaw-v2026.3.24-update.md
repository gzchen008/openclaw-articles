# OpenClaw v2026.3.24 更新：Teams 大升级、技能管理重做、安全修复

> 发布日期：2026年3月25日 | 当前版本：2026.3.13 → 2026.3.24

OpenClaw v2026.3.24 带来了大量改进，涵盖 Microsoft Teams 全面升级、技能管理系统重构、安全沙箱修复，以及多个频道的稳定性提升。以下是详细更新内容。

---

## 🎯 亮点更新

### 1. Microsoft Teams 全面升级

Teams 频道迁移到官方 SDK，带来了完整的 AI 代理体验：

- **流式 1:1 回复**：对话不再干等，实时看到 AI 思考过程
- **欢迎卡片**：带快捷提示词的引导卡片，新用户上手更友好
- **打字指示器**：AI 正在回复时显示输入状态
- **消息编辑和删除**：支持修改和撤回已发送的消息
- **原生 AI 标签**：界面中清晰标识 AI 代理身份

### 2. 技能管理系统重构

Control UI 的技能页面焕然一新：

- **状态筛选标签**：全部 / 就绪 / 需要设置 / 已禁用，一目了然
- **一键安装依赖**：coding-agent、gh-issues、weather 等内置技能支持自动安装缺失依赖
- **详情弹窗**：点击技能卡片展开需求、开关、API Key 配置等完整信息
- **"需要设置"替代"缺失"**：更友好的提示文案，附带 Key 获取指引

### 3. Gateway OpenAI 兼容性增强

新增 `/v1/models` 和 `/v1/embeddings` 端点，支持在 `/v1/chat/completions` 和 `/v1/responses` 中转发模型覆盖参数。这意味着更多 RAG 工具和客户端可以直接对接 OpenClaw Gateway。

### 4. 安全修复

- **沙箱媒体绕过修复**：关闭了 `mediaUrl`/`fileUrl` 别名绕过，防止工具和消息操作逃出媒体根目录限制
- **出站媒体策略对齐**：严格按 fs 策略控制文件访问

## 📋 完整变更列表

### 代理与工具

- `/tools` 命令现在显示当前代理实际可用的工具
- Control UI 新增实时"当前可用"工具展示区
- 代理工作区文件支持懒加载 Markdown 预览
- 模型选择器新增"未设置"占位提示

### CLI 与容器

- 新增 `--container` 参数，支持在运行中的 Docker/Podman 容器内执行命令
- Node 22 最低版本降至 22.14+（推荐仍为 Node 24）
- `openclaw update` 更新前预检目标版本 Node 要求

### 频道改进

- **Slack**：恢复富文本回复、自动渲染 Options 为按钮/选择器
- **Discord**：支持 AI 生成线程名、超时可见回复
- **WhatsApp**：修复群组回声抑制和 reply-to-bot 检测
- **Telegram**：修复论坛话题 #General 路由、照片上传失败、403 永久失败处理

### macOS 应用

- 配置页面改为可折叠树形侧边栏
- API Key 编辑器增加"获取密钥"链接和存储路径提示

### 其他修复

- 网关重启后通过心跳唤醒中断的会话
- 频道启动隔离，单个频道故障不再阻塞其他频道
- Docker 新安装不再因网络命名空间问题失败
- 嵌入式运行的 SecretRef 未解析不再崩溃
