📱 移动端大升级

━ Android：聊天设置 UI 重构 ━

全新的聊天设置界面，更现代、更易用：

• 优化了布局和交互

• 修复了 HttpURLConnection 内存泄漏

• 使用 Google Code Scanner 扫描二维码（更稳定）

━ iOS：新增欢迎向导 ━

首次打开有引导页了：

• 帮助新用户快速上手

• 展示核心功能

• 更友好的 onboarding 体验

💬 Slack 交互式回复

终于来了！ Slack 频道支持交互式回复指令。

之前：只能被动回复消息
现在：可以主动发起交互，支持按钮、菜单等交互元素

使用场景：

• 审批请求（一键通过/拒绝）

• 选择操作（多个选项按钮）

• 表单填写（弹出对话框）

🐳 Docker 改进

━ 时区支持 ━

docker run -e OPENCLAW_TZ=Asia/Shanghai openclaw/gateway

终于可以设置时区了！定时任务、日志时间都能正确显示。

━ 安全修复 ━

防止 Gateway Token 泄露：

• 修复了 Docker 构建上下文中的 token 泄露风险

• 更安全的构建流程

━ 系统更新 ━

所有 Dockerfile 都添加了 apt-get upgrade，确保基础镜像安全。

🌐 浏览器控制增强

━ 批量操作支持 ━

browser.batch([

{ kind: "click", ref: "e1" },

{ kind: "type", ref: "e2", text: "hello" },

{ kind: "press", key: "Enter" }

])

一次性执行多个操作，减少往返延迟。

━ 会话生命周期加固 ━

• 现有会话驱动验证更严格

• 会话生命周期管理更健壮

• 防止会话异常断开

⚡ 性能翻倍

修复 2x 内存回归：

• Plugin-SDK chunks 去重

• 内存占用降低约 50%

• 更流畅的运行体验

🪟 Windows 改进

抑制控制台窗口闪烁：

之前重启 OpenClaw 时会弹出黑色控制台窗口
现在：完全静默，后台运行

🍎 macOS 改进

━ 防止误杀 Docker Desktop ━

修复了 PortGuard 在远程模式下误杀 Docker Desktop 的问题。

━ 尊重审批设置 ━

exec-approvals.json 配置在 Gateway prompter 中生效了。

📨 飞书改进

━ 非ASCII 文件名支持 ━

之前：上传"报告.pdf"可能乱码
现在：完美支持中文文件名

━ 防止重复回复 ━

添加早期事件级去重，彻底解决飞书群聊重复回复问题。

🔄 会话管理

━ 保留关键状态 ━

重置会话时保留：

• lastAccountId：最后使用的账号

• lastThreadId：最后的线程 ID

━ 保持角色连续性 ━

压缩摘要时保留：

• 人设（persona）

• 语言（language）

Agent 不会"忘记"自己是谁了。

⏰ Cron 改进

防止嵌套死锁：

修复了隔离 cron 嵌套 lane 死锁问题，定时任务更稳定。

🛡️ 其他安全修复

• Telegram：媒体传输策略防SSRF

• Gateway：限制未响应的客户端请求数量

• UI：保持共享认证的不安全连接
