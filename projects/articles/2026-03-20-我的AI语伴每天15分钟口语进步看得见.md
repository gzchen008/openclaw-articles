## 实战：3 个高频场景

### 场景 1：日常对话练习

**目标**：提升口语流利度

**方法**：
- 每天选一个话题（天气、工作、兴趣爱好）
- 用目标语言和 AI 聊 10-15 分钟
- 让 AI 纠正你的语法和用词

**示例对话**：

```
你: Today weather is good, I want go outside.

AI: Almost right! Here's a more natural way:
"Today the weather is good, I want to go outside."

Tips:
- Add "the" before "weather"
- Add "to" before "go"
```

### 场景 2：面试模拟

**目标**：准备英语面试

**方法**：
- 让 AI 扮演面试官
- 模拟常见问题（自我介绍、优势劣势、职业规划）
- 让 AI 给出评分和改进建议

**Prompt 模板**：

```
你是一位严格的英语面试官，请问我 5 个常见的面试问题，每个问题后给我的回答打分（1-10分）并指出可以改进的地方。
```

### 场景 3：旅游场景

**目标**：准备出国旅游

**方法**：
- 让 AI 扮演酒店前台、餐厅服务员、出租车司机
- 模拟点餐、问路、入住等场景
- 学习实用的旅游用语

**示例**：

```
你：我想要一个海景房，有吗？

AI（扮演前台）：
Certainly! We have ocean-view rooms available. 
Would you prefer a room with a balcony?
```

## 进阶技巧：让 AI 语伴更聪明

### 技巧 1：设置语言水平

告诉 AI 你的水平，它会调整难度：

```
我是英语中级水平，请用简单的词汇和句子和我对话，不要太复杂。
```

### 技巧 2：请求详细反馈

对话结束后，让 AI 总结：

```
请总结我今天对话中的 3 个语法错误，并给出正确的表达。
```

### 技巧 3：词汇积累

遇到好用的表达，让 AI 解释：

```
"Make ends meet" 是什么意思？能造个句吗？
```

### 技巧 4：发音练习

如果 OpenClaw 接入了语音功能，可以：

- 发语音消息给 AI
- AI 会纠正你的发音
- 还能听到标准发音
