## 🚀 核心新功能

### 1️⃣ CLI 备份命令（全新）

现在可以用命令行备份和恢复 OpenClaw 配置：

```bash
# 创建备份
openclaw backup create

# 仅备份配置
openclaw backup create --only-config

# 不含工作区
openclaw backup create --no-include-workspace

# 验证备份
openclaw backup verify
```

**使用场景**：
- 升级前备份
- 迁移到新机器
- 破坏性操作前自动提示

### 2️⃣ Talk 模式静音超时

语音对话时，可以配置静音多久后自动发送：

```json
{
  "talk": {
    "silenceTimeoutMs": 2000
  }
}
```

**默认值**：各平台保持原有默认值
**自定义范围**：1000-10000 毫秒

**效果**：说完话后，系统自动等待你设定的静音时长，然后发送语音转文字结果。

### 3️⃣ Brave 搜索 LLM 增强模式

Brave 搜索现在支持 LLM 增强模式，返回带来源元数据的摘要片段：

```json
{
  "tools": {
    "web": {
      "search": {
        "brave": {
          "mode": "llm-context"
        }
      }
    }
  }
}
```

**优势**：
- ✅ 返回结构化摘要
- ✅ 包含来源链接
- ✅ 适合 AI 消费

---

### 4️⃣ ACP 来源追踪

ACP（Agent Communication Protocol）现在支持来源追踪：

```bash
openclaw acp --provenance off|meta|meta+receipt
```

**模式说明**：
- `off` - 不追踪
- `meta` - 保留来源元数据
- `meta+receipt` - 显示会话追踪 ID

**使用场景**：多 Agent 协作时追踪消息来源

---
