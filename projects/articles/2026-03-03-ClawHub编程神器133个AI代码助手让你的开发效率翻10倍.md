## ⚠️ 避坑指南

### 1. 选择合适的 Agent

- **简单任务**：用 **coding-agent**（单 Agent）
- **复杂项目**：用 **claude-team** 或 **cc-godmode**（多 Agent）
- **安全要求高**：用 **local-first-llm**（本地模型）

### 2. 本地模型配置

- **Ollama**：适合 Mac/Linux，安装简单
- **LM Studio**：适合 Windows，有图形界面
- **llamafile**：单文件部署，最简单

### 3. 成本控制

- 优先用本地模型（免费）
- 云端 API 按需使用
- 设置每日限额，避免超支

### 4. 安全注意事项

- 不要让 AI 直接操作生产环境
- Code Review 后仍需人工确认
- 敏感代码不要发送到云端 API
