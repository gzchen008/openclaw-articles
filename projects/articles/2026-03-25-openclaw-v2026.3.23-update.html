OpenClaw 2026.3.23 发布：安全加固、UI 重构与 50+ 项修复

从 2026.3.13 到 2026.3.23，OpenClaw 带来了重大更新：CSP 安全加固、全新 Control UI 主题、多平台稳定性修复，以及针对开发者体验的全面优化。

🔒 安全与合规

CSP 内联脚本白名单

Control UI 现在自动计算所有内联脚本的 SHA-256 哈希值，并将其加入 CSP script-src 指令。这意味着：

  • 默认阻止所有内联脚本
  • 仅允许经过哈希验证的引导代码
  • 完全符合现代安全标准

执行审批安全增强

Shell 包装器的白名单匹配现在更加严格：

  • 拒绝单引号的 $0/$n 令牌
  • 禁止换行符分隔的 exec
  • 仅接受 exec -- 载体形式

🎨 Control UI 重构

全新 Knot 主题

  • 配色方案：黑红配色，WCAG 2.1 AA 级对比度
  • 按钮系统：统一 btn--icon、btn--ghost、btn--xs 原语
  • 圆角控制：用离散档位替代滑块
  • 无障碍：全面添加 aria-label

配置图标增强

Diagnostics、CLI、Secrets、ACP、MCP 等配置区域现在拥有专属图标，导航更加直观。

🤖 模型支持更新

Qwen（阿里云 ModelStudio）

新增标准按量付费端点：

  • 中国区 DashScope 端点
  • 全球区 Qwen API 端点
  • 保留原有的 Coding Plan 端点
  • Provider 组重命名为 Qwen (Alibaba Cloud Model Studio)

Mistral 模型修复

  • 降低 bundled Mistral 最大 token 默认值到安全输出预算
  • openclaw doctor --fix 现在可以修复旧的 Mistral 配置
  • 解决 422 拒绝错误

OpenAI Codex OAuth

支持环境配置的 HTTP/HTTPS 代理，解决代理环境下的 OAuth 刷新问题。

🔧 50+ 项 Bug 修复

核心修复

  • Auth：OpenAI token 保存不再回退到过期值
  • Plugins：恢复 bundled plugin runtime 打包
  • CLI：channels login/logout 自动选择单通道
  • ClawHub：macOS 认证路径修复，技能浏览不再 429
  • Browser：Chrome MCP attach 流程稳定性提升
  • Gateway：锁冲突不再导致 crash-loop

平台特定修复

  • Telegram：DM topics 线程上下文修复
  • Discord：特权命令返回明确的未授权回复
  • LINE：jiti 启动时的符号重定义错误
  • Matrix：运行时 API 导出冲突
  • WhatsApp：doctor --fix 不再写入无效的 allowlist

📦 安装与升级

全局安装最新版本：

npm install -g openclaw@latest

验证版本：

openclaw --version

应显示：OpenClaw 2026.3.23
