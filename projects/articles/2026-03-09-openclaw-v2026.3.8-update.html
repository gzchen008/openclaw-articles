OpenClaw v2026.3.8 更新：备份命令 + LLM 搜索 + Talk 静音超时

发布时间：2026-03-09

📦 更新总览

OpenClaw 发布了最新版本 v2026.3.8，带来了 5 大新功能和多项 Bug 修复。

🚀 核心新功能

1️⃣ CLI 备份命令（全新）

现在可以用命令行备份和恢复 OpenClaw 配置：

# 创建备份
openclaw backup create

# 仅备份配置
openclaw backup create --only-config

# 验证备份
openclaw backup verify

💡 使用场景：升级前备份、迁移配置、定期归档

2️⃣ Talk 静音超时配置

语音模式现在可以配置静音多久后自动发送：

{
  "talk": {
    "silenceTimeoutMs": 2000
  }
}

默认各平台不同，现在可以自定义了！

3️⃣ Brave 搜索 LLM 增强模式

返回带来源元数据的结构化摘要：

{
  "tools": {
    "web": {
      "search": {
        "brave": {
          "mode": "llm-context"
        }
      }
    }
  }
}

✅ 结构化摘要 + ✅ 来源链接 + ✅ 适合 AI 消费

4️⃣ ACP 来源追踪

多 Agent 协作时追踪消息来源：

openclaw acp --provenance meta+receipt

三种模式：off | meta | meta+receipt

5️⃣ macOS 远程网关配置

macOS 应用现在支持直接配置远程网关 Token，改善远程模式体验。
