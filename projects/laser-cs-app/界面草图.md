## 设计规范

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 背景色 | `#1a1a2e` | 深色主题 |
| 卡片背景 | `#16213e` | 次深色 |
| 强调色 | `#00d9ff` | 霓虹蓝 |
| A阵营 | `#e84118` | 红色 |
| B阵营 | `#0077ff` | 蓝色 |
| 胜利 | `#4cd137` | 绿色 |
| 失败 | `#e84118` | 红色 |
| MVP | `#ffd700` | 金色 |

### 字体规范

| 用途 | 大小 | 字重 |
|------|------|------|
| 标题 | 20-36px | Bold |
| 正文 | 14-16px | Regular |
| 辅助文字 | 12px | Regular |
| 数字 | 18-28px | Bold |

### 交互规范

| 操作 | 反馈 |
|------|------|
| 按钮点击 | 缩放 + 颜色变化 |
| 设备连接 | 状态动画（扫描中→已连接） |
| 击杀事件 | 震动 + 音效 + 屏幕提示 |
| 胜利/失败 | 全屏动画 + 音效 |

### 屏幕尺寸

- 设计尺寸：375 x 812 px（iPhone X）
- 适配范围：iPhone SE ~ iPhone 15 Pro Max

## 运动相机功能说明

### 预录制流程

```
游戏开始
    ↓
APP 发送「预录制开始」指令
    ↓
运动相机循环缓存最近 N 秒（5-30秒可配置）
    ↓
APP 检测到击杀事件
    ↓
APP 发送「快门指令」到运动相机
    ↓
运动相机保存预录制片段 + 继续录制 M 秒（5-10秒）
    ↓
完成精彩镜头录制
```

### 支持的相机类型

| 类型 | 连接方式 | 延迟 | 说明 |
|------|----------|------|------|
| GoPro | 蓝牙/WiFi | < 100ms | 第三方运动相机 |
| Insta360 | 蓝牙/WiFi | < 100ms | 第三方运动相机 |
| 自研相机 | WiFi (TCP/UDP) | < 50ms | 低延迟定制方案 |

### 精彩镜头管理

| 功能 | 说明 |
|------|------|
| 自动保存 | 击杀敌人时自动保存 |
| 手动保存 | 被击杀时可选保存 |
| MVP 高光 | 本局 MVP 自动生成集锦 |
| 分享 | 支持分享到社交平台 |
