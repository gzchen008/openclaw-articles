### 📝 详细介绍

Cline 是一个**基于 Claude Sonnet 的智能编程助手**，直接集成在 VS Code 中。与传统的代码补全工具不同，Cline 能够：

**核心功能：**
- ✅ **文件操作** - 自动创建、编辑文件，修复语法错误
- ✅ **终端命令** - 在你的终端中执行命令并监控输出
- ✅ **浏览器交互** - 启动 headless 浏览器，点击、输入、滚动、截图
- ✅ **项目分析** - 分析文件结构和 AST，快速理解大型项目
- ✅ **MCP 支持** - 使用 Model Context Protocol 扩展自身能力
- ✅ **人机协作** - 每一步操作都需你批准，安全可控

### 🚀 快速开始

**1️⃣ 安装（VS Code）**
```
打开 VS Code → 扩展市场 → 搜索 "Cline" → 安装
```
或直接访问：https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev

**2️⃣ 配置 API Key**
- 安装后点击侧边栏的 Cline 图标
- 输入你的 Claude API Key（或其他支持的模型）
- 开始对话即可

**3️⃣ 使用示例**
```
💬 "帮我把这个 React 组件重构为 TypeScript"
💬 "分析一下这个 bug，看看是什么问题"
💬 "根据这个设计图（拖拽图片）创建一个登录页面"
```
