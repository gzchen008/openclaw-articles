## 📦 产品定义（选定方向）

### 🎯 产品名称：「小红书文案AI」—— 专注小红书的爆款文案生成器

**一句话介绍**：3分钟生成10条小红书爆款笔记，让素人也能写出万赞文案

**目标用户**：
- **核心用户**：小红书博主（1000-10万粉丝），每月发5-20篇笔记
- **次要用户**：品牌方运营、MCN机构内容团队、电商卖家
- **痛点**：写文案耗时、不知道什么能火、缺乏灵感、不会写标题
- **付费意愿**：个人创作者￥19.9/月，专业博主￥49.9/月，企业客户￥299/月起

**为什么选择小红书**：
- 小红书月活2.6亿，创作者数量超5000万
- 小红书文案有独特「语法」（emoji多、分段、口语化）
- 平台算法鼓励高频更新，内容需求巨大
- 变现路径清晰（带货、接广告、知识付费）
- 相比抖音视频，文案创作门槛更低，更适合AI生成

### 🚀 核心功能模块

#### 模块 1：智能文案生成（核心）

**功能 1.1：主题输入 → 批量生成小红书笔记**
- 用户输入：产品/主题 + 笔记类型（种草/测评/干货/故事/探店）+ 目标人群
- AI生成：10条小红书风格笔记（每条配标题+正文+标签+emoji）
- 小红书特色：
  - 标题：20字内，带数字/悬念/痛点，用「｜」分隔
  - 正文：分段清晰，每段2-3行，多用emoji分隔
  - 标签：3-5个核心标签+2-3个长尾标签
  - 表情：每段至少1个emoji，增加亲和力
- 示例：
  ```
  输入：「护肤精华，主打抗老」+ 小红书 + 种草风
  输出：
  1. 「25+抗老必入！这瓶精华让我告别了细纹...」+ 正文 + #抗老精华 #护肤分享
  2. 「成分党狂喜！深挖这瓶精华的抗老秘密...」+ 正文 + #成分党 #精华测评
  ...共10条
  ```

**功能 1.2：爆款结构模板**
- 内置100+爆款文案结构：
  - 「痛点+解决方案+效果见证」
  - 「数字吸引+清单体+行动号召」
  - 「故事引入+情感共鸣+产品植入」
  - 「权威背书+专业分析+购买建议」
- 用户可选择结构，AI按模板生成

**功能 1.3：风格模仿学习**
- 用户粘贴喜欢的爆款文案
- AI分析其风格（用词、节奏、 emoji使用）
- 生成同风格的新文案

#### 模块 2：多平台适配

**功能 2.1：一键多平台改写**
- 输入一条内容，自动生成：
  - 小红书版（emoji多、分段清晰、标签丰富）
  - 抖音版（口语化、钩子在前、引导互动）
  - 公众号版（深度、结构完整、阅读流畅）
  - 微博版（简短、话题标签、适合传播）

**功能 2.2：平台特性优化**
- 小红书：自动添加「｜」「✨」等符号，优化首图标题
- 抖音：前3秒钩子设计，引导评论话术
- 公众号：小标题、金句框、阅读节奏优化

#### 模块 3：智能优化工具

**功能 3.1：爆款度检测**
- AI评分：预测文案的爆款潜力（1-100分）
- 优化建议：哪些词可以换、哪里加emoji、开头如何改

**功能 3.2：违禁词/敏感词检测**
- 自动检测平台违禁词（如小红书不能说的「最」「第一」等）
- 提供替代表达建议

**功能 3.3：标签推荐**
- 根据内容自动生成标签组合
- 热门标签推荐（结合当下热点）
- 长尾标签策略（避开大竞争）

#### 模块 4：内容管理（付费功能）

**功能 4.1：文案库**
- 保存生成的所有文案
- 分类管理（按主题、平台、日期）
- 快速搜索和复用

**功能 4.2：内容日历**
- 规划未来7/30天的发布内容
- 定时生成提醒
- 一键导出发布计划

**功能 4.3：数据分析（高级版）**
- 接入用户的小红书/抖音数据（需授权）
- 分析哪类文案表现最好
- AI学习优化生成策略

#### 3. 小红书算法核心机制

**CES 评分模型**：
```
CES = 点赞×1 + 收藏×2 + 评论×4 + 转发×5 + 关注×8
```
- 收藏权重 > 点赞（内容有用性）
- 评论权重高（互动质量）
- 关注转化率最高（粉丝沉淀）

**流量池阶梯**：
| 流量池 | 曝光量 | 关键 |
|--------|--------|------|
| 初始池 | 200-500 | 新笔记默认进入 |
| 小流量池 | 1000-5000 | CES > 阈值 |
| 中流量池 | 5000-1万 | 持续高互动 |
| 大流量池 | 1-10万 | 爆款潜质 |
| 全站推荐 | 10万+ | 现象级 |

**关键洞察**：
1. **冷启动是关键**：前30分钟黄金时间，帮助用户过初始池
2. **标签匹配**：3-5核心标签 + 2-3长尾标签
3. **账号权重**：原创度、垂直度、活跃度、互动率、合规度

#### 4. 小红书文案「语法」总结

**标题公式（高点击率）**：
- 数字法：「3个月减20斤｜上班族的减脂餐攻略」
- 悬念法：「别再这样护肤了！90%的人都做错了」
- 痛点法：「熬夜脸黄怎么办？这瓶精华救了我」
- 对比法：「从120斤到95斤｜我的蜕变记录」
- 清单法：「2024必买的5支口红｜黄皮显白」

**正文结构**：
```
第1段：钩子（3秒抓住）
第2-4段：正文（每段2-3行，emoji开头）
第5段：互动引导（提问/号召）
```

#### 5. 产品功能与算法结合点

| 功能 | 算法价值 |
|------|----------|
| 违禁词检测 | 避免限流 |

#### 3. 小红书算法核心机制

**CES 评分模型**：
```
CES = 点赞×1 + 收藏×2 + 评论×4 + 转发×5 + 关注×8
```
- 收藏权重 > 点赞（内容有用性）
- 评论权重高（互动质量）
- 关注转化率最高（粉丝沉淀）

**流量池阶梯**：
| 流量池 | 曝光量 | 关键 |
|--------|--------|------|
| 初始池 | 200-500 | 新笔记默认进入 |
| 小流量池 | 1000-5000 | CES > 阈值 |
| 中流量池 | 5000-1万 | 持续高互动 |
| 大流量池 | 1-10万 | 爆款潜质 |
| 全站推荐 | 10万+ | 现象级 |

**关键洞察**：
1. **冷启动是关键**：前30分钟黄金时间，帮助用户过初始池
2. **标签匹配**：3-5核心标签 + 2-3长尾标签
3. **账号权重**：原创度、垂直度、活跃度、互动率、合规度

#### 4. 小红书文案「语法」总结

**标题公式（高点击率）**：
- 数字法：「3个月减20斤｜上班族的减脂餐攻略」
- 悬念法：「别再这样护肤了！90%的人都做错了」
- 痛点法：「熬夜脸黄怎么办？这瓶精华救了我」
- 对比法：「从120斤到95斤｜我的蜕变记录」
- 清单法：「2024必买的5支口红｜黄皮显白」

**正文结构**：
```
第1段：钩子（3秒抓住）
第2-4段：正文（每段2-3行，emoji开头）
第5段：互动引导（提问/号召）
```
