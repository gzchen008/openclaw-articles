## 📋 系统概述

### 核心定位
基于大语言模型 + 量化分析 + 多因子模型的AI选股系统，帮助投资者从海量信息中挖掘投资机会，提供买入/持有/卖出的智能决策建议。

### 解决的问题
1. **信息过载** - 每天数千条财报、新闻、研报，人工无法全面分析
2. **情绪化交易** - 投资者容易受市场情绪影响做出非理性决策
3. **分析效率低** - 传统基本面分析耗时，错过最佳买卖时机
4. **缺乏系统策略** - 个人投资者没有科学的选股和风控体系

### 核心价值主张
> "让AI成为您的24小时投资分析师，用数据驱动投资决策"

## 🏗️ 系统架构

### 1. 数据层

#### 1.1 市场数据
- **实时行情**：股价、成交量、换手率、资金流向
- **历史数据**：K线数据（日线/周线/月线）、历史市盈率
- **技术指标**：MACD、KDJ、RSI、布林带、均线系统
- **板块数据**：行业轮动、板块资金流向、龙头股表现

#### 1.2 基本面数据
- **财务报表**：财报关键指标（营收、净利润、ROE、毛利率）
- **估值指标**：PE、PB、PS、PEG、股息率
- **成长指标**：营收增长率、净利润增长率
- **质量指标**：资产负债率、现金流、应收账款周转

#### 1.3 另类数据
- **舆情数据**：社交媒体情绪、股吧讨论热度
- **新闻数据**：公司公告、行业新闻、政策变化
- **研报数据**：券商评级、目标价、盈利预测
- **宏观数据**：利率、CPI、PMI、汇率

### 2. 分析引擎层

#### 2.1 量化分析引擎
```python
# 多因子选股模型
factors = {
    'value_factors': ['PE', 'PB', 'PS', '股息率'],      # 价值因子
    'growth_factors': ['营收增长率', '净利润增长率'],      # 成长因子
    'quality_factors': ['ROE', '毛利率', '现金流'],       # 质量因子
    'momentum_factors': ['1月涨幅', '3月涨幅', 'RSI'],   # 动量因子
    'sentiment_factors': ['舆情得分', '资金流入']         # 情绪因子
}
```

#### 2.2 AI分析引擎
- **财报解读**：LLM自动分析财报，提取关键变化和风险点
- **新闻情绪分析**：实时分析新闻情绪，判断利好/利空
- **研报摘要**：自动总结研报核心观点和投资建议
- **事件驱动分析**：识别催化剂事件（业绩预告、政策发布）

#### 2.3 技术分析引擎
- **趋势识别**：自动识别上涨/下跌/震荡趋势
- **形态识别**：头肩顶/底、双底、三角形等经典形态
- **支撑压力**：计算关键支撑位和压力位
- **量价分析**：识别放量突破、缩量回调等信号

### 3. 策略层

#### 3.1 选股策略
| 策略名称 | 策略逻辑 | 适用场景 |
|---------|---------|---------|
| **价值挖掘** | 低估值+高股息+稳定盈利 | 熊市、震荡市 |
| **成长猎手** | 高成长+行业景气+机构增持 | 牛市早期 |
| **动量追踪** | 强势股+资金流入+突破形态 | 趋势行情 |
| **事件驱动** | 业绩预增+政策利好+重组预期 | 事件窗口期 |
| **量化多因子** | 多因子打分+排序选股 | 全天候 |

#### 3.2 择时策略
- **大盘择时**：基于MACD、均线系统判断市场趋势
- **板块轮动**：识别当前热点板块和轮动节奏
- **个股择时**：结合技术形态和资金流向确定买卖点

#### 3.3 风控策略
- **仓位管理**：根据市场环境动态调整仓位（30%-100%）
- **止损机制**：固定止损（-8%）+ 移动止损（跌破20日均线）
- **止盈机制**：目标止盈（+20%）+ 趋势止盈（跌破10日均线）
- **分散投资**：单只股票仓位不超过20%，行业分散

#### 4.1 每日选股池

```
【风险提示】⚠️
- 300XXX 某某生物：业绩不及预期，建议回避
```

#### 4.2 持仓监控
- **实时盈亏**：每只股票的盈亏状况和收益率
- **调仓建议**：何时加仓/减仓/清仓
- **风险预警**：触及止损线、异常波动提醒

#### 4.3 市场洞察
- **每日复盘**：大盘分析、热点板块、资金流向
- **周报/月报**：策略回顾、收益归因、改进建议

## ⚠️ 风险提示

### 系统局限性
1. **历史不代表未来** - 回测表现好不代表实盘能盈利
2. **黑天鹅事件** - 无法预测突发重大事件（如疫情、战争）
3. **模型过拟合** - 参数优化可能导致过度拟合历史数据
4. **数据质量** - 财务数据造假、延迟会影响分析结果

### 使用建议
1. AI选股作为**参考工具**，不是**唯一决策依据**
2. 建议先用**模拟盘**验证策略有效性
3. **严格风控**，单只股票不超过20%仓位
4. **持续学习**，理解系统逻辑再实盘
