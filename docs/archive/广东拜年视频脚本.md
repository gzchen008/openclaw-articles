# 🧧 广东恭喜发财拜年视频脚本

**视频时长**：15-20 秒
**风格**：喜庆、热闹、有广东味
**音乐**：广东贺年歌曲（如《财神到》《迎春花》）

---

## 📍 场景设置

**背景**：
- 家中客厅，贴有挥春（春联）
- 桌上摆：金桔盆栽、利是封（红包）、全盒（糖果盒）
- 可选：小醒狮摆件、桃花/水仙花

**服装**：
- 红色/金色系新衣服
- 可戴红色围巾或帽子

---

## 🎬 分镜脚本

### 【0-3秒】开场 · 醒狮动作

**镜头**：中景 → 推近

**动作**：
- 双手举起，做醒狮头动作（两手在头顶，手指张开）
- 左右摇摆 2-3 次
- 面带笑容，眼睛有神

**音效**：锣鼓声（咚咚锵）

**字幕**：🦁 醒狮贺岁

---

### 【3-8秒】主祝福 · 粤语吉利话

**镜头**：近景（半身）

**动作**：
- 双手抱拳作揖（左手抱右手）
- 轻微鞠躬 2 次
- 保持笑容

**台词**（粤语）：
> "恭喜发财！🧧
> 身体健康，龙马精神！
> 财源滚滚来，利是逗来！💰
> 添福添寿，万事胜意！✨"

**字幕**：
```
恭喜发财！
身体健康 龙马精神
财源滚滚来 利是逗来
添福添寿 万事胜意
```

---

### 【8-12秒】派利是动作

**镜头**：特写 → 中景

**动作**：
- 从口袋/桌上拿出利是封（红包）
- 双手持利是，向前递出
- 点头微笑

**台词**（粤语）：
> "利利是是，掂过碌蔗！🎋
> 拿到利是，笑口常开！"

**字幕**：
```
利利是是 掂过碌蔗
拿到利是 笑口常开
```

---

### 【12-15秒】收尾 · 金桔祝福

**镜头**：中景（带金桔）

**动作**：
- 走到金桔盆栽旁
- 轻抚金桔叶子
- 指向金桔

**台词**（粤语）：
> "金桔满堂，大吉大利！🍊
> 新年快乐，阖家幸福！"

**字幕**：
```
金桔满堂 大吉大利
新年快乐 阖家幸福
```

---

### 【15-18秒】结束画面

**镜头**：远景

**动作**：
- 双手比心 / 挥手
- 或者双手抱拳再次作揖
- 大声喊："恭喜发财！"

**字幕**：
```
🧧 恭喜发财 🧧
[你的名字] 敬上
```

**音乐**：渐弱结束

## 💡 拍摄建议

### 广东特色元素（可选加入）

1. **茶楼场景**：
   - 手拿点心笼（虾饺/烧卖）
   - 台词："饮茶食包，笑口常开！🥟"

2. **醒狮互动**：
   - 如果有小醒狮道具，可以举着舞动
   - 台词："醒狮贺岁，鸿运当头！🦁"

3. **年花背景**：
   - 站在桃花/水仙前
   - 寓意："花开富贵，桃花滚滚来！🌸"

### 粤语吉利话库（可替换）

- **学业**："学业进步，步步高升！📚"
- **事业**："步步高升，事业顺景！💼"
- **财运**："财源广进，黄金满屋！💰"
- **健康**："身壮力健，龙精虎猛！💪"
- **家庭**："一家团圆，和气生财！👨‍👩‍👧‍👦"

### 动作要点

✅ **抱拳作揖**：左手抱右手（左为尊），手心向内
✅ **派利是**：双手递出，表示尊重
✅ **笑容**：自然、真诚，不要太僵硬
✅ **语速**：中等偏快，有活力
