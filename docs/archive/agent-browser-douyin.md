🚀 快速上手：
1️⃣ 安装：npm install -g agent-browser
2️⃣ 打开网页：agent-browser open https://xxx.com

**文案：**
🤔 浏览器自动化工具怎么选？

今天对比 3 个热门工具：

1️⃣ **Playwright**
- 微软出品
- 功能全面
- 但配置复杂

2️⃣ **Puppeteer**
- Google 出品
- Chrome 专用
- 学习曲线陡

3️⃣ **agent-browser** ⭐
- Vercel 出品
- Rust 编写，超快
- 专为 AI Agent 设计
- 命令简单，上手快

🔥 我的推荐：
- 新手 → agent-browser
- 企业级 → Playwright
- Chrome 深度用户 → Puppeteer
