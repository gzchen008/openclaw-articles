## 制作要点

### 视觉风格
- **配色**: Claude 品牌色（橙色 + 紫色）
- **字体**: 思源黑体 / PingFang SC
- **动画**: 流畅、科技感、不花哨

### 音频
- **背景音乐**: 轻快科技风（BPM 120-140）
- **音效**: 打字声、成功提示音、转场音效
- **语速**: 1.2x 倍速（紧凑不拖沓）

### 字幕
- **位置**: 底部居中
- **样式**: 白色字体 + 黑色描边
- **大小**: 占屏幕宽度 70%

### 时长控制
- **抖音**: 60 秒版本（精简核心）
- **B站**: 90-120 秒版本（完整演示）
- **小红书**: 60 秒版本 + 图文笔记
