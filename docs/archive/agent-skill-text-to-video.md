## 制作建议

### 文生视频工具推荐
- **Runway Gen-2**：高质量，支持长视频
- **Pika Labs**：免费，易于使用
- **Stable Video Diffusion**：开源，可本地运行
- **Sora**（OpenAI）：效果最好（需等待开放）

### 提示词模板
```
A tutorial video about Claude Agent Skill. 
Scene: [场景描述]
Style: Tech tutorial, clean UI, animated graphics
Duration: [时长]
Text overlay: [屏幕文字]
```

### 音频建议
- **背景音乐**：科技感电子乐，BPM 120-140
- **音效**：打字声、成功提示音、转场音效
- **配音**：可使用 AI 配音工具（ElevenLabs、Azure TTS）
