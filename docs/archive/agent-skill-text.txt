以前每次都要粘贴一堆要求，现在写个 Skill 文件，AI 自己就会看！

创建一个文件夹，写个 skill.md 文件，就这么简单！

Agent Skill 有三层设计，越深层越省 Token！

第一层是元数据层，始终加载，包含名称和描述。

第二层是指令层，按需加载，就是 skill.md 的内容。

第三层是资源层，条件触发，包含 Reference 和 Script。

比如提到钱，就自动查财务规定，这就是 Reference 的威力！
